## 核心理念：场景 = 世界观的物理容器

> 场景设计的终极目标不是"好看"，而是让场景本身成为叙事工具——
> 它承载情绪、定义角色所处的世界、并开放给观众进行二创。

**场景的三重功能：**
1. **叙事锚点**：每一个场景名都是 `@标签`，是 AI 生成的稳定参考点
2. **情绪容器**：场景的光影/构图/色调直接决定观众的情绪体验
3. **IP 宇宙入口**：好的场景，观众会拿去做二创（"雪山湖里"就是一个世界观符号，不是背景图）

---

## 场景设计的 AI 时代范式转变

| 旧范式 | 新范式 |
|--------|--------|
| 场景 = 背景板 | 场景 = 世界观前期容器 |
| 场景服务于角色 | 场景与角色共同叙事 |
| 追求精美写实 | 追求情绪辨识度 |
| 场景固定不变 | 场景开放，供观众/AI 二创 |
| 正面全景图够用 | 需要多角度、多光效变体 |

**关键认知**：设定稿中的场景描述 = 写给 AI 的提示词。场景设定越精准，后续视频生成的氛围越可控，越不需要每次重新描述。

## 场景板标准规范（主体库三板之场景板）

> 来源：白梦客知识库 `wiki/AI视频创作/Vidu/主体库大师.md`

| 字段 | 标准 |
|------|------|
| 场景名 | 简洁有辨识度，可作为 `@标签` 引用 |
| 出场集数 | 首次出场集数 |
| 场景描述 | 200字以上：时间段、地点、空间布局、光影基调、氛围 |
| 文生图描述词 | 150字以上，结构：`主体环境 + 光影参数 + 视觉风格 + 情感氛围` |

### 场景描述词结构公式

```
主体环境（地点/空间类型）
+ 时间/天气（决定自然光色温）
+ 空间逻辑（人物动线/机位可能性）
+ 光影参数（主光类型 + 介质 + 光比）
+ 氛围关键词（情绪方向）
+ 风格标签（画风/导演风格参考）
```

### 场景设计三原则

| 原则 | 说明 |
|------|------|
| **空间逻辑** | 场景布局需符合人物动线和摄影机位设置 |
| **时代/世界观特征** | 场景装饰、建筑风格需准确反映世界观背景 |
| **光影优先** | 场景光照设计需考虑时间、天气、情绪氛围需求 |

## 场景诊断能力：从叙事需求到视觉实现

> 好的场景不是"好看"，而是"有效"。场景诊断是确定**需要什么场景**的能力，然后才是**如何实现**。

### 核心框架：三层分析

```
叙事需求
    ↓
第一层：这个场景要完成什么叙事任务？
    ↓
第二层：完成这个任务需要什么空间特性？
    ↓
第三层：这些特性如何转化为视觉语言？
```

---

### 第一层：叙事任务分类

每个场景本质上只做这几件事之一（或同时做几件）：

| 叙事任务 | 本质问题 | 例子 |
|---------|---------|------|
| **建立世界观** | 这个世界是什么样的？ | 展示外星城市 |
| **揭示角色** | 这是一个什么样的人？ | 展示角色住所 |
| **建立关系** | 角色之间是什么关系？ | 两人在什么空间对话 |
| **制造张力** | 故事在积累什么？ | 暴风雨前的平静 |
| **释放情绪** | 观众应该感受到什么？ | 孤独、温暖、压迫 |
| **转折标记** | 故事发生了什么变化？ | 同一个房间但变了 |
| **收束主题** | 故事在说什么？ | 主题物件再次出现 |

---

### 第二层：从任务到空间特性

```
叙事任务 ──→ 需要什么空间特性
━━━━━━━━━━━━━━━━━━━━━━━━━━━

【揭示角色】
→ 需要私密空间（卧室、书房、个人物品）
→ 需要有"生活痕迹"（未整理、习惯物）
→ 需要有"反常"（不该在这里的东西在这里）

【建立权力关系】
→ 需要有物理高度差（谁在高处）
→ 需要有入口位置（谁是闯入者）
→ 需要有核心物品的位置（谁是中心）

【制造张力】
→ 需要"即将发生什么"的感觉
→ 需要有"不对"的细节（颜色、秩序、光）
→ 需要空间"容纳"即将发生的事

【释放孤独】
→ 需要"空"的视觉
→ 需要有"人的痕迹"但人不在
→ 需要封闭但不能压抑（不是监禁感）

【建立世界观】
→ 需要足够多的新元素让世界观可信
→ 但不能太满（那是另一个任务）
→ 需要"日常感"来锚定陌生感
```

### 第三层：视觉语言转化

同样的叙事任务，不同的实现路径：

```
叙事：揭示"角色是一个孤独的人"
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

路径A（环境叙事）：
空公寓，但有生活痕迹
→ 牙刷只有一支、水池里一个碗、沙发上有人坐过的凹陷
→ 视觉语言：空旷、细节、人的缺席

路径B（光影叙事）：
灯火通明的大公寓，但只有一盏台灯亮着
→ 空间很大但人只在很小范围活动
→ 视觉语言：光域划分、光的孤立

路径C（物品叙事）：
普通公寓，但有大量反常物品
→ 跑鞋很贵但从未跑步、酒柜满满但都是同一种酒
→ 视觉语言：物品揭示行为模式

路径D（时间叙事）：
场景本身正常，但有"时间感"
→ 浴室镜子上有水渍（刚洗完澡）、床铺是温的（刚起床）
→ 视觉语言：时间暗示、连续性
```

### 空间特性 × 情绪效果表

| 空间特性 | 情绪效果 | 适合叙事 |
|---------|---------|---------|
| 封闭+压抑 | 窒息、危险、压迫 | 恐惧、紧张、幽闭 |
| 封闭+舒适 | 安全、庇护、温暖 | 亲密、休息、归属 |
| 开放+空旷 | 孤独、自由、渺小 | 失落、成长、抑郁 |
| 开放+繁华 | 活力、可能性、眩晕 | 兴奋、机会、迷失 |
| 私密+空 | 孤独、沉思、抑郁 | 内心、反省、思念 |
| 私密+有痕 | 温暖、安全、归属 | 亲密、回忆、稳定 |
| 公共+陌生 | 不安、警惕、好奇 | 探索、新开始 |
| 公共+熟悉 | 归属、日常、放松 | 社交、身份、习惯 |

### 光影特性 × 叙事

| 光影类型 | 情绪效果 | 视觉特征 |
|---------|---------|---------|
| 硬光+顶光 | 审讯、压迫、无处躲藏 | 头顶直射、阴影在眼窝 |
| 柔光+侧面 | 回忆、 nostalgia、温柔 | 斜光、golden hour |
| 背光+轮廓 | 孤独、神秘、分离 | 剪影、看不清脸 |
| 局部光（单点） | 聚焦、孤立、唯一 | 黑暗中的单点亮 |
| 无光（纯黑） | 恐惧、未知、虚无 | 全黑、只能想象 |
| 频闪/不稳定 | 紧张、危险、不稳定 | 闪烁、忽明忽暗 |
| 冷光（蓝） | 冷漠、孤独、疏离 | 蓝色调、没有温度 |
| 暖光 | 温暖、亲密、安全 | 黄橙、居家感 |
| 自然光 | 真实、时间感、开放 | 时间标记、天光 |

### 物品的叙事功能

```
物品类型 ──→ 叙事功能
━━━━━━━━━━━━━━━━━━━━━━

【日常物品的不正常状态】
→ 牙刷只有一支（独居/关系断裂）
→ 水池有碗（刚吃过/懒得收拾）
→ 床铺半边整齐（假装的正常）
→ 衣服没洗/堆着（生活失序）

【不应该在这里的物品】
→ 枪出现在温馨场景（危险即将发生）
→ 药瓶出现在年轻人场景（秘密/问题）
→ 别人的物品在私人空间（第三者）

【物品的位置关系】
→ 主角的杯子vs配角的杯子（距离）
→ 夫妻照不在卧室（在客厅显眼位置=关系是表演）
→ 孩子的东西不在家里（失踪/失去）

【物品的数量和密度】
→ 空旷=孤独/死亡/离开
→ 密集=压迫/混乱/失控
→ 刚好=正常/平衡
```

### 诊断检查清单

拿到一个叙事需求时，逐项检查：

```
□ 1. 我知道这个场景的核心叙事任务是什么吗？
□ 2. 这个任务是这个场景唯一/主要要完成的吗？
□ 3. 我选择的空间类型是否服务于这个任务？
□ 4. 我选择的视觉元素是否都在强化同一个叙事？
□ 5. 有没有元素在"说别的事情"？（冲突的信号）
□ 6. 这个场景的情绪底色是否统一？
□ 7. 光影是否在帮助叙事而不是干扰？
□ 8. 场景之后是什么？有没有考虑节奏？
□ 9. 观众需要知道什么？场景给了足够信息吗？
□ 10. 场景说太多了吗？有没有留白？
```

### 案例分析模板

拿到一个叙事需求时，用这个流程分析：

**练习：暴发户的新办公室**

```
1. 核心任务：揭示角色（同时建立关系）
   → 表面想建立权力，实际在暴露不安全感

2. 空间选择：
   → 不能是纯粹的权力展示（太直白）
   → 选择了：新到过分的欧式，但混着奇怪的东西

3. 视觉方向：
   【选错了时代的家具】
   → 路易十四式书桌（但真正的贵族不会用这么夸张的）
   
   【过于崭新的物品】
   → 皮沙发：没有使用痕迹，崭新得像是道具
   → 书柜：书脊都是朝外的，但排列整齐得像没读过
   
   【尺寸和比例的"用力过度"】
   → 办公桌太大了（真正有权的人不需要大桌子来证明）
   → 椅子太夸张了（真正自信的人不需要这么张扬的座椅）
   
   【反常的细节（揭示自卑）】
   → 座机电话位置不对（在角落，像是不太会用）
   → 电脑是最新款但屏幕背对门口（怕被看到在工作？）
   → 没有任何私人物品（没有照片、没有纪念品）
   → 但有一面墙专门展示"荣誉"
```

**AI提示词方向：**

```
"newly decorated executive office, mix of Louis XIV desk and leather sofa,
uncomfortable juxtaposition of European palace style and Chinese armchair,
all furniture looks brand new, ornate curtain, oversized chair, massive desk,
empty bookshelf with perfectly aligned book spines,
reception area with wall of trophies and framed photos,
harsh overhead fluorescent light, no warm lamp light,
mood: ostentatious yet hollow, trying too hard, wealth without taste"
```

### 常见错误图鉴

```
错误1：任务模糊
━━━━━━━━━━━━━━━━
"我要做一个家庭场景"
→ 什么类型的家庭？什么时刻？什么情绪？
→ 失败标志：所有家庭场景看起来都一样

错误2：元素堆砌
━━━━━━━━━━━━━━━━
"书房就要有：书架+书+台灯+椅子+地球仪+..."
→ 每个元素都应该有理由出现
→ 失败标志：去掉任何一个元素场景感觉不变

错误3：太过直白
━━━━━━━━━━━━━━━━
"表现孤独，就放一个哭泣的人"
→ 应该让观众自己发现
→ 失败标志：情绪写在脸上而不是空间里

错误4：孤立设计
━━━━━━━━━━━━━━━━
"这个场景很好！但前后接不上"
→ 场景应该和故事其他部分有呼应
→ 失败标志：场景单独看不错但串联时断裂

错误5：过度解释
━━━━━━━━━━━━━━━━
"这里有个台灯，因为主角曾经有个温暖的童年"
→ 视觉不应该解释自己
→ 失败标志：需要文字说明才能理解

错误6：信号冲突
━━━━━━━━━━━━━━━━
"我想表现温馨，但用了蓝色冷光"
→ 视觉信号不一致
→ 失败标志：观众感受到的不是你想表达的
```

### 叙事需求 × 场景类型映射表

| 叙事需求 | 优先场景类型 | 空间特性 | 视觉方向 |
|---------|------------|---------|---------|
| 建立世界观 | 公共空间、城市 | 陌生元素+日常锚点 | 新元素够信服 |

| 收束主题 | 主题物件出现 | 首尾呼应 | 物件回归 |

## 分镜命名与场景锚定规则

> 来源：白梦客知识库 `wiki/AI视频创作/Vidu/分镜生成系统.md`

### @标签引用规范

在分镜描述中，场景必须用 `@标签` 锚定：

```
分镜描述格式：
在【时间描述】的【@场景名】中，[@角色名] 做了什么...

示例：
在瓢泼大雨的夜晚，【@废弃剧场】中，[@小丑] 缓缓张开双臂...
```

**场景绘图提示词格式：**
```
@场景名：（室内/室外广角，建立镜头）场景的详细视觉描述...
```

### 禁止代词规则

所有分镜必须是**可独立理解的、自包含的原子化指令**。
- ❌ 禁止：「他站在那里」
- ✅ 要求：「[@角色名] 站在 [@场景名] 的中央」

## 建筑风格 × 情绪速查表

> 精确的建筑风格词汇是场景提示词的基础。无论动画还是写实，风格名称决定了 AI 输出的空间气质。

### 西方建筑风格

| 风格 | 年代 | 视觉特征 | 情绪语言 | AI 提示词 |
|------|------|---------|---------|----------|
| **古希腊** | BC 8C-1C | 柱式（多立克/爱奥尼/科林斯）、三角楣、比例严谨 | 理性、崇高、秩序 | `Greek temple, Doric columns, marble, symmetrical` |
| **古罗马** | BC 1C-5C | 拱券、穹顶、混凝土、宏大公共空间 | 权力、永恒、帝国感 | `Roman architecture, arched vault, grand colonnade` |
| **哥特** | 12C-16C | 尖拱、飞扶壁、彩窗玻璃、高耸尖塔 | 敬畏、神秘、升腾 | `Gothic cathedral, pointed arches, stained glass, ribbed vault` |
| **文艺复兴** | 14C-17C | 圆顶、对称立面、古典柱式复兴、人体比例 | 和谐、人文、理想化 | `Renaissance palazzo, dome, symmetrical facade, classical order` |
| **巴洛克** | 17C-18C | 曲线、装饰繁复、壁画穹顶、明暗戏剧 | 戏剧性、奢华、狂欢 | `Baroque interior, ornate ceiling fresco, dramatic chiaroscuro` |
| **洛可可** | 18C | 贝壳曲线、粉彩、金箔、极致装饰 | 轻盈、甜腻、放纵 | `Rococo salon, pastel colors, gilded ornament, shell motifs` |
| **新古典** | 18C-19C | 回归严谨柱式、对称、简化装饰 | 庄严、理性回归、公共性 | `Neoclassical building, grand columns, pediment, clean lines` |
| **工业革命** | 19C | 铸铁结构、玻璃幕墙、火车站式大跨度 | 进步、机械、力量 | `Industrial architecture, iron framework, glass roof, factory` |
| **新艺术** | 1890-1910 | 有机曲线、植物纹样、铸铁装饰 | 自然、浪漫、颓废 | `Art Nouveau, organic curves, floral ironwork, Mucha style` |
| **装饰艺术** | 1920-1940 | 几何图案、金属光泽、阶梯式轮廓 | 摩登、奢华、速度 | `Art Deco, geometric patterns, chrome, stepped silhouette` |
| **包豪斯/现代主义** | 1920-1970 | 几何纯净、白墙平顶、钢玻璃混凝土 | 理性、极简、乌托邦 | `Bauhaus, flat roof, white walls, glass curtain wall, minimal` |
| **粗野主义** | 1950-1975 | 清水混凝土、巨型体块、裸露结构 | 压迫、力量、反乌托邦 | `Brutalist concrete, raw exposed structure, massive block` |
| **后现代** | 1970-1990 | 拼贴历史符号、色彩对撞、反讽装饰 | 讽刺、游戏、多义 | `Postmodern architecture, colorful, ironic classical reference` |
| **解构主义** | 1980- | 倾斜扭曲、碎片化、非线性 | 不安、挑战、动态 | `Deconstructivist, tilted walls, fragmented forms, Gehry style` |
| **参数化** | 2000- | 流线型曲面、算法生成形态 | 未来、有机科技 | `Parametric architecture, flowing curves, Zaha Hadid style` |

### 东方建筑风格

| 风格 | 视觉特征 | 情绪语言 | AI 提示词 |

## 空间比例与情绪心理学

> 空间的尺度直接操控观众的身体感知——这是比色调、光影更底层的情绪工具。

### 天花板高度 × 情绪

| 高度 | 体感 | 情绪效果 | 适用情境 |
|------|------|---------|---------|
| < 2.4m | 低矮压迫 | 焦虑、窒息、私密 | 地下室、牢房、密室逃脱 |
| 2.4m~3m | 普通住宅感 | 日常、亲切、安全 | 家庭场景、办公室 |
| 3m~5m | 开阔商业感 | 正式、公共、疏离 | 大堂、画廊、教室 |
| 5m~10m | 挑高仪式感 | 崇敬、仪式、权力 | 教堂、法院、剧院 |
| > 10m | 穹顶巨构 | 渺小、敬畏、神圣 | 大教堂、宫殿、太空站 |

| 空间类型 | 体感 | 情绪效果 | AI 关键词 |

## 场景诊断能力 × 从叙事到视觉（中英双语）

> 核心框架：给定一个叙事目标 → 判断需要什么场景 → 确定场景的视觉方向
> Core Framework: Narrative Goal → Scene Type → Visual Direction

### 诊断三问 / Diagnostic Three Questions

```
□ 1. 这个场景的核心叙事任务是什么？
   What is the core narrative task?
   （建立世界观/揭示角色/建立关系/制造张力/释放情绪/转折标记/收束主题）
   (establish world / reveal character / build relationship / create tension / release emotion / mark turning point / close theme)

□ 2. 完成这个任务需要什么空间特性？
   What spatial qualities are needed?
   （私密/公共、高/低、开放/封闭、暖/冷）
   (private/public, high/low, open/closed, warm/cold)

□ 3. 这些特性如何转化为视觉语言？
   How to translate into visual language?
   （光影/材质/色彩/秩序/细节）
   (lighting/material/color/order/detail)
```

### 诊断检查清单 / Diagnostic Checklist

```
□ 1. 我知道这个场景的核心叙事任务是什么吗？
   Do I know the core narrative task?
□ 2. 这个任务是这个场景唯一/主要要完成的吗？
   Is this the primary task for this scene?
□ 3. 我选择的空间类型是否服务于这个任务？
   Does the space type serve the task?
□ 4. 我选择的视觉元素是否都在强化同一个叙事？
   Do all visual elements reinforce the same narrative?
□ 5. 有没有元素在"说别的事情"？（冲突的信号）
   Any element "saying something else"? (conflicting signals)
□ 6. 这个场景的情绪底色是否统一？
   Is the emotional tone unified?
□ 7. 光影是否在帮助叙事而不是干扰？
   Does lighting help rather than hinder narrative?
□ 8. 场景之后是什么？有没有考虑节奏？
   What comes after? Is rhythm considered?
□ 9. 观众需要知道什么？场景给了足够信息吗？
   What does audience need to know? Enough info given?
□ 10. 场景说太多了吗？有没有留白？
   Too much said? Any negative space?
```

## 景深系统 × 焦点控制与叙事（中英双语）

> 景深 = 画面中"清晰"的范围。控制景深就是控制观众"看到什么"和"忽略什么"。
> Depth of Field = the "sharp" range in frame. Controlling DoF controls what audience sees and ignores.

### 景深三要素 / Three Elements of DoF

```
【光圈与景深】Aperture → DoF
光圈越小（f值越大）→ 景深越深 → 虚化越弱
光圈越大（f值越小）→ 景深越浅 → 虚化越强
Small aperture (high f) → deep DoF → weak blur
Large aperture (low f) → shallow DoF → strong blur

【焦距与虚化】Focal Length → Blur
焦距越长 → 景深越浅 → 虚化越强
焦距越短 → 景深越深 → 虚化越弱
Long focal length → shallow DoF → strong blur
Short focal length → deep DoF → weak blur

【距离与虚化】Distance → Blur
拍摄距离越近 → 景深越浅 → 虚化越强
拍摄距离越远 → 景深越深 → 虚化越弱
Closer to subject → shallow DoF → strong blur
Farther from subject → deep DoF → weak blur
```

## 风格融合逻辑（中英双语）

> 融合不是"叠加"，而是"有机结合"。好的融合=1+1>2，坏的融合=两层皮。
> Fusion is not "layering" but "organic combination". Good fusion = 1+1>2, bad = two separate layers.

### 融合公式 / Fusion Formula

```
融合 = 找共同点 + 找张力点 + 决定主次
Fusion = Find common ground + Find tension points + Decide hierarchy

Step 1: 确定A风格的核心特征
Step 2: 确定B风格的核心特征
Step 3: 找到可以并存的特征
Step 4: 找到产生张力的特征
Step 5: 决定谁是主、谁是次
Step 6: 用次风格来挑战主风格
```

### 融合类型 / Fusion Types

| 类型 Type | 逻辑 Logic | 效果 Effect | 梦的模型 Chinese | Midjourney English |
|---------|----------|-----------|------------------|-------------------|

| **功能融合** | 神圣+日常/权力+私密 | 反讽、深度 | `功能融合，神圣日常，权力私密反讽` | `functional fusion, sacred meets mundane, power privacy irony` |

## 尺度控制（中英双语）

> AI出图常见问题：建筑大人小、比例混乱。控制尺度=给AI明确的参照物。
> AI common issue: architecture too big, humans too small. Scale control = give AI clear references.

### 尺度感知原理 / Scale Perception Principles

```
人眼的尺度判断基于：
1. 参照物（熟悉物体的大小）
2. 材质细节（材质和尺度的关系）
3. 光影比例（阴影长度）
4. 空间关系（物体之间的距离）

Human eye judges scale based on:
1. Reference objects (familiar objects' size)
2. Material details (relationship between material and scale)
3. Light/shadow ratio (shadow length)
4. Spatial relationships (distance between objects)
```

## 建筑构成法则（中英双语）

> 建筑的视觉力量来自构成关系——对称/不对称、虚实、韵律、比例。
> 理解这些法则，才能在场景设计中主动**创造**空间感，而不只是复制参考图。
> Architectural visual power comes from compositional relationships — symmetry/asymmetry, void/solid, rhythm, proportion.
> Only by understanding these laws can you actively *create* spatial sense in scene design, not just copy reference images.

### 对称与不对称（中英双语）/ Symmetry & Asymmetry

| 构成方式 Composition | 视觉效果 Visual | 情绪语言 Emotional | 使用场景 Scene | 梦的模型 Chinese | Midjourney English |
|-------------------|----------------|-------------------|--------------|------------------|-------------------|
| **轴对称** | 庄严、稳定、权威 | 秩序、神圣、压迫 | 宫殿正面、教堂中殿、法院大厅 | `轴对称，庄严稳定权威，秩序神圣压迫` | `bilateral symmetry, solemn stable authoritative, order sacred oppressive` |
| **近似对称** | 整体有序但有呼吸感 | 和谐、生活化、可亲 | 传统民居、中式院落 | `近似对称，整体有序有呼吸感，和谐生活可亲` | `near symmetry, overall ordered with breathing room, harmonious livable approachable` |
| **刻意不对称** | 动态、张力、不安 | 现代、挑战、自由 | 解构建筑、废墟场景 | `刻意不对称，动态张力不安定，现代挑战自由` | `deliberate asymmetry, dynamic tension unstable, modern challenging free` |
| **局部对称+整体不对称** | 复杂但有章法 | 城市感、层次丰富 | 街景、混合建筑群 | `局部对称整体不对称，复杂有章法，城市感丰富` | `local symmetry global asymmetry, complex structured, urban layered` |

**场景设计原则（中英双语）/ Scene Design Principles：**

| 需求 Need | 方法 Method | 梦的模型 Chinese | Midjourney English |
|----------|-----------|------------------|-------------------|
| 表达**秩序/权力** | 用对称 | `表达秩序权力，用对称` | `express order/power, use symmetry` |
| 表达**自由/混乱** | 打破对称 | `表达自由混乱，打破对称` | `express freedom/chaos, break symmetry` |

### 虚与实（开口与实墙）（中英双语）/ Void & Solid (Openings & Solid Walls)

> 建筑立面的「虚」（窗、门、开口）和「实」（墙体、柱子）的比例，直接控制空间的呼吸感。
> The ratio of "void" (windows, doors, openings) to "solid" (walls, columns) directly controls the breathing feel of space.

| 虚实比 Ratio | 视觉效果 Visual | 情绪 Emotion | 典型建筑 Example | 梦的模型 Chinese | Midjourney English |
|-------------|----------------|-------------|-----------------|------------------|-------------------|

### 极简留白/负空间 × 中英双语 / Minimalist Negative Space

> 大面积负空间——东方美学「计白当黑」，西方极简主义同样推崇。
> 关键不在于留白本身，而在于**主体与空白的比例关系**。
> Large negative space — Eastern "treat white as black", Western minimalism also advocates.
> Key is not the white space itself, but the **ratio between subject and empty space**.

| 留白比例 Ratio | 情绪效果 Emotion | 适用情境 Scene | 梦的模型 Chinese | Midjourney English |
|---------------|----------------|--------------|------------------|-------------------|
| 40%~50% | 呼吸感、舒适、开放 | 日常、舒缓 | `留白40-50%，呼吸舒适开放` | `negative space 40-50%, breathing comfortable` |
| 50%~60% | 沉思、内省、安静 | 独白、冥想 | `留白50-60%，沉思内省安静` | `negative space 50-60%, contemplative meditative` |
| 60%~70% | 孤独、疏离、仪式感 | 告别、孤立、哲思 | `留白60-70%，孤独疏离仪式` | `negative space 60-70%, lonely isolated ceremonial` |
| 70%~80% | 极简、压迫、虚无 | 极端情感、高端品牌 | `留白70-80%，极简压迫虚无` | `negative space 70-80%, minimalist oppressive void` |

### 演员表演空间 × 为角色留位

> 核心原则：**场景是演员的舞台**——即使画面中没有人，构图、光影、空间关系都在为演员预留位置。
> 画面不是"没有人"，而是"人即将进入"或"人刚离开"。

#### 空间分配的三维预留

```
【地面空间】演员站立/移动的区域
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
画面中要预留演员的"站位"
→ 演员从哪里走进画面？
→ 演员主要的活动区域在哪？
→ 演员和其他物件的关系？

【视线空间】演员看向哪里的空间
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
演员的视线方向要留白
→ 演员在看什么？（物件/窗户/门/另一个角色）
→ 视线方向要有东西可看
→ 视线尽头要有"答案"

【光区预留】演员会站在哪
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
主光区 = 演员最可能出现的位置
→ 场景中的主光区就是演员的"主舞台"
→ 道具摆放要考虑演员进入后的光影效果
→ 不要让演员的表演"破坏"已有的光影逻辑
```

#### 构图中的表演预留

```
【为对话预留】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
双人中景需要预留两个人的空间
→ 两人之间有呼吸感
→ 两人的视线在画面中交汇
→ 中间可以有物件但不能阻断关系

【为情绪预留】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
特写需要预留面部空间
→ 面部朝向的空间（headroom）
→ 不要让物件"切"脸
→ 视线方向留白

【为运动预留】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

### 构图应用原则

1. **层次优先**：先建立前景-中景-背景三层
2. **留白克制**：宁多勿少，保持画面呼吸感
3. **情绪物体**：选择有象征意义的细节
4. **风格自由融合**：从不同来源提取技法组合，创造你自己的视觉语言
5. **真实感检查**：前景材质是否真实自然？散景是否不规则？光影是否符合物理逻辑？

### 色彩基础理论 / Color Theory Basics

```

- Hue: what color
- Value: how bright/dark
- Chroma: how pure/grey

## 场景串联 × 分镜逻辑（中英双语）

> 单个场景优秀不等于分镜优秀。场景之间的串联决定了整体叙事的节奏和情绪走向。
> Great individual scenes ≠ great sequence. Scene connections determine pacing and emotional trajectory.

### 场景串联三原则 / Three Principles of Scene Linking

```
1. 情绪梯度：场景之间情绪要有关联/递进/对比
2. 视觉节奏：远/中/近景的搭配要有呼吸感
3. 信息流：新信息 → 确认信息 → 反转信息

1. Emotional gradient: scenes connect via relationship/progression/contrast
2. Visual rhythm: wide/medium/close-up ratio needs breathing room
3. Information flow: new info → confirm info → twist info
```

| 连接类型 Type | 逻辑 Logic | 效果 Effect | 梦的模型 Chinese | Midjourney English |

### 六、设计原则应用指南

#### 如何使用这些原则

1. **选择基础**：确定你的世界观基调（如：东方幻想、末日废土、现代都市）
2. **提取原则**：从上述原则表中选择2-3个核心原则
3. **组合混搭**：不同来源的原则可以自由组合（如：宋代曲线美学+黑暗之魂的尺度压迫）
4. **生成提示词**：将原则转化为AI提示词语言

#### 原则混搭示例

```
# 示例1：东方幻想 + 游戏美学
宋代曲线屋顶 + 黑暗之魂的永恒衰败 + 艾尔登法环的神圣诅咒
= 仙侠遗迹：废弃的宋代宫殿，屋顶曲线优雅但已崩塌，
  残留金色装饰散发着被诅咒的微光

# 示例2：现代都市 + 电影美学
东方明珠的垂直韵律 + 布达佩斯大饭店的对称强迫症 + 盗梦空间的失重
= 梦境都市：一座超现实的摩天大楼，每层都是完美的对称结构，
  但建筑在失重中旋转

# 示例3：末日废土 + 流浪地球
死亡搁浅的线缆叙事 + 流浪地球的宏大工程 + 时间冻结美学
= 后末日重建：地球发动机废墟之间，人类用残余的电缆
  重新连接被冰封的城市
```

## 速查：场景设计关键决策点

| 决策 | 推荐做法 |
|------|---------|
| 场景需要传达什么情绪？ | 先定情绪方向，再选光影/构图参数 |
| 光影选哪种？ | 从五种基础光影类型中选择，按世界观自由组合 |
| 构图层次怎么建立？ | 前景遮挡 + 极简留白 + 情绪细节点缀，比例自由 |
| 场景描述给 AI 多少字？ | 文生图描述词 150 字以上，含：空间/材质/光影/色调/情感 |
| 场景在分镜中如何引用？ | 用 `@场景名` 标签，禁止代词，保持原子化 |
| 如何防止场景 AI 输出随机性？ | 先出场景板设定图，锁定后作为参考图输入 |
| 场景只要一张正面图吗？ | 不够，需四镜（标准/情感/动态/氛围）覆盖不同叙事需求 |
| 场景和角色如何配合？ | 场景色彩应与角色色块形成对比/互补，避免视觉相融 |
| 世界观是全新的怎么办？ | 从建筑风格表和材质表中自由组合，混搭创造新世界 |
