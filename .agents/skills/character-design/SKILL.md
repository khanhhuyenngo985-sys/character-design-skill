## 第一章：核心范式

### 旧范式 vs 新范式

| 旧范式 | 新范式 |
|--------|--------|
| 让角色讨人喜欢 | 让角色**令人难忘** |
| 帅哥靓女 | 不完美、有缺陷、有争议 |
| 单一原型 | **原型混搭**（杀手+导师+爱人同一人） |
| 完美主角团 | **群像设计**，每个配角都可能成为爆点 |
| 近似色、和谐配色 | **色彩暴力**（高饱和、极端对比、色彩隔离） |
| 固定姿势站桩 | **姿态叙事**（成长弧线、状态变化） |
| 动画帧数平均分配 | **时间即喜剧**（关键帧集中在笑点爆发） |

**核心原理**：角色的价值完全取决于他与其他角色的**差异性**，而非自身的完美。

**当代记忆锚点 = 情感刺痛 + 文化共鸣 + 打脸美学**

近年令人印象深刻的角色（电锯人、星期三、Jinx、Lucy），无一例外都带有某种**被压抑的暴力感**或**边缘属性**。这种"难忘"来自于：
- **情绪上的强烈冲击**：爽剧的打脸、逆袭、不按常理出牌
- **欲望的宣泄出口**：角色替观众完成现实中不敢做的反抗
- **复杂矛盾的人格**：强大与脆弱并存，冷酷与母性共生

### 媒介革命与角色设计范式迭代

> 每次媒介革命 = 人与内容关系的重组 = 角色设计要求的升级

| 时代 | 观众与媒介的关系 | 角色设计要求 |
|------|----------------|-------------|
| 电视时代 | 被动接受，按播出表守候 | 清晰、讨喜、辨识度高即可 |
| PC/网剧时代 | 可暂停、回放、截图、弹幕讨论 | 需要经得起细看的细节和记忆锚点 |
| 手机/短剧时代 | 随时随地、全天候、碎片化 | 首秒抓眼、角色辨识度极强 |
| **AI时代** | **观众成为创作者，可二创、可互动** | **角色必须足够开放，容纳多重解读** |

**核心结论**：今天的角色设计，不是给"被动观众"设计的，而是给**主动共创者**设计的。

## 第二章：角色DNA七字诀

> 设计角色的七个维度。全部用"差异性"贯穿。

### 一、群像优先

**核心**：先设计群像，再提取个体。角色A的"善良"必须由角色B的"邪恶"来衬托。

**受众推导链**：

```
目标受众 → 他们的审美偏好 → 他们见过什么 → 他们期待什么 →
什么元素能让他们"意外" → 反差设计方向
```

**DNA原则**（防止AI融合）：

```
发型：高/矮 → 短发/长发 → 颜色差异（黑/金/红）
体型：高/矮 → 胖/瘦 → 肩宽/窄
脸型：圆/方/椭圆/长
肤色：白/黄/黑/棕
服装：职业装/休闲/制服
色彩：暖/冷/对比/单色
```

**AI多人相融问题预防（在设计阶段）：**
1. **色块差异**：每个主要角色分配专属色彩标识，确保视觉远看可区分
2. **体型差异**：高矮胖瘦、身型轮廓要有明显差别
3. **服装差异**：材质感、复杂程度、风格标签各不相同
4. **姿态差异**：驼背 vs 挺拔，展开 vs 收缩

### 二、原型混搭

**八种原型**：

| 原型 | 核心功能 | AI设计优先级 |
|------|---------|--------------|
| **Hero（英雄）** | 推动叙事弧线 | 成长弧线 |
| **Shadow（阴影）** | 制造最大冲突 | 威胁感 |
| **Mentor（导师）** | 给予智慧/道具 | 标志性细节 |
| **Trickster（诡计者）** | 打破规则，制造意外 | 姿态反差 |
| **Threshold Guardian（阈限守卫）** | 测试Hero | 弱点设计 |
| **Herald（信使）** | 宣告/召唤 | 视觉信号 |
| **Shapeshifter（变形者）** | 制造悬念 | DNA锚点 |
| **Ally（伙伴）** | 提供支持 | 独立性 |

**原型叠加**：两个原型叠加在同一角色上，差异性翻倍。

```
Hero + Shadow = 被使命撕裂的悲剧英雄（艾伦·耶格尔）
Hero + Mentor = 用死亡教育后辈的牺牲者（欧比旺）
Shadow + Ally = 表面敌人实际帮助主角（三角头）
Mentor + Shadow = 反差型导师，平日是普通老头（老乞丐/火云邪神）
```

### 三、姿态叙事

**姿态公式**：

```
姿态 = 重心位置 × 四肢角度 × 脊椎曲直 × 头部位置
```

**三原则**：
1. **重心**：高重心=自信（直立），低重心=压抑（下沉）
2. **四肢**：张开=开放，紧缩=封闭
3. **脊椎**：直=力量，弯=谦卑/疲惫

**成长弧线姿态变化**：

```
阶段1：驼背（重心不稳）→ 阶段2：直立（找到方向）→ 阶段3：略微僵硬（机甲覆盖人性）
```

### 四、情绪-表情-姿态联动（必修）

情绪、表情、姿态是三个互相咬合的齿轮。AI视频里必须让三者同时出现在Prompt中。

**八种核心情绪联动矩阵**：

| 情绪 | 面部变化 | 姿态变化 |
|------|---------|---------|
| **愤怒** | 眉心皱缩/嘴角下压/颏肌紧绷 | 身体前倾/拳头紧握/重心前移 |
| **悲伤** | 眉尾上翘/嘴角下弯/眼睑下垂 | 肩膀内收/身体下沉/重心偏下 |
| **恐惧** | 眼窝张大/眉毛抬起/嘴唇微张 | 身体后退/双臂护胸/重心后移 |
| **惊讶** | 眉毛抬起/眼睛张大/O型嘴 | 身体僵住/双手展开/重心居中 |
| **厌恶** | 眉毛下压/眼睛微眯/嘴角上翘 | 身体后仰/头部后移/偏头 |
| **轻蔑** | 眉毛不对称/眯眼/嘴角单边上扬 | 身体后仰/单手插兜/偏头 |
| **喜悦** | 眉毛上扬/眼睛弯月形/嘴角上扬 | 身体放松/肩膀抬起/重心居中 |
| **平静** | 眉毛水平/眼睛正常/嘴角横线 | 身体放松/重心居中/肌肉松弛 |

**愤怒三型（最常用）**：

```
压抑型：嘴角抿成直线，拳头握紧垂在身侧
爆发型：川字眉，牙齿咬紧，身体前倾双拳举起
威压型：眉心无皱纹，嘴角微微下压，身体静止不动
```

### 五、色彩暴力

**三种类型**：

| 类型 | 手法 | 案例 |
|------|------|------|
| 极端饱和 | 超高饱和度 | 瑞贝卡：绿红配 |
| 极端对比 | 黑白 vs 彩虹 | 星期三：黑白 vs 室友五彩 |
| 色彩隔离 | 环境单色+角色点缀 | 夜之城：全深蓝+大卫黄色夹克 |

**色彩心理学**：

| 色调 | 情感 | 使用场景 |
|------|------|---------|
| 暖色（红/橙/黄） | 热情、人性、温暖 | 主角、恋爱情节 |
| 冷色（蓝/紫） | 神秘、理性、距离 | 杀手、科学家 |
| 高饱和 | 强烈、刺激、年轻 | 反英雄、边缘角色 |
| 低饱和/灰调 | 复杂、成熟、压抑 | 成人向、复杂角色 |
| 黑色 | 克制、死亡、权威 | 反英雄、丧服 |
| 白色 | 纯洁、脆弱、空虚 | 天真、死亡前期 |

### AI视角：色彩稳定性研究（重点）

> 为什么高饱和色彩在AI视频里更稳定？哪些色彩容易"糊"？

**为什么AI对高饱和色更稳定**：

```
1. 边缘检测友好
   高饱和色 → 像素值偏离均值大 → 边缘算子响应强
   → 主体轮廓在特征图上更突出

2. 注意力机制偏向
   Transformer的Self-Attention优先响应高方差区域
   高饱和像素方差大 → 自然获得更多注意力权重

3. 色彩一致性损失
   训练数据中，高饱和区域被优先保护
   低饱和区域（灰褐/雾/阴影）更容易被模糊处理
```

**色彩稳定性光谱表**：

| 色彩类型 | 稳定性 | 备注 |
|---------|--------|------|
| 纯红/纯黄/霓虹蓝 | ★★★★★ | AI最优先，红黄是视觉极点 |
| 深红/橙色/电紫 | ★★★★ | 饱和但有深色压底 |
| 正蓝/绿色 | ★★★ | 蓝饱和但偏冷，绿在自然中太多 |
| 深灰 | ★★ | 低饱和，需强光影辅助 |
| 肤色/浅灰 | ★★ | 中间调，最易模糊 |
| 白色/雾效果 | ★ | 无饱和度，完全依赖光影 |

**🔴 高危场景（避开或加强光影）**：

```
1. 沙漠/沙尘 → 肤色融进沙地 → 解决：红/蓝围巾+强光轮廓
2. 雪景/雾景 → 浅色服装消失 → 解决：深色或饱和色
3. 夜店昏暗 → 面部模糊 → 解决：霓虹光打脸/反光材质
4. 森林草木 → 绿棕角色像长在树上 → 解决：红橙色服装
5. 赛博雨夜 → 边缘崩塌 → 解决：霓虹服装+局部打光
```

**五种AI稳定色彩结构**：

```
1. 主体高饱和 + 背景中性灰      ← 最安全基础结构
2. 主体高饱和 + 背景互补色      ← 最强视觉冲击
3. 主体深色 + 背景高亮度        ← 明度差异而非饱和度
4. 主体低饱和 + 背景高饱和      ← 反向（黑白主角vs高饱和幻觉）
5. 主体单色 + 配饰高饱和锚点    ← 最实用：灰褐角色+黄色围巾
```

**⚠️ 最大误区：色彩暴力 ≠ 全画面高饱和**
→ 正确：结构化对比 = 主体饱和 + 环境低饱和（或互补）
→ 全身高饱和 + 背景也高饱和 = AI不知道该突出谁

**实战检查清单**：

```
□ 角色主色饱和度 ≥ 背景饱和度？
□ 角色色彩和环境色彩有足够HSV色相差？
□ 有光源方向辅助主体/背景边缘分离？
□ 肤色角色避开了灰色/白色/浅色背景？
□ 有"色彩锚点"（高饱和局部细节）？
□ 雾/雨/夜间场景有强局部光源？
□ 色彩对比用在了叙事转折点？
```

### 六、服装道具

**四大功能**：
1. **造型**：人为制造尖锐/圆润边缘，改变剪影
2. **叙事**：承载角色成长弧线
3. **身份认同**：纹样代表门派/阵营/文化
4. **锚定**：固定道具让AI记住角色

**服装即角色符号**：

| 角色 | 色彩 | 符号意义 |
|------|------|---------|
| 麦克斯 | 红围巾 | 唯一的暖色 = 人性最后的痕迹 |
| 星期三 | 黑白 | 死亡/哥特/与周围隔绝 |
| 大卫 | 黄色夹克 | 母亲遗物 = 人性/温暖 |

### 七、极端对比

**七种极端对比**：

```
1. 外表 vs 真实（普通人外表/绝世高手内心）
2. 色彩 vs 内容（可爱画风/黑暗故事）
3. 姿态 vs 身份（杀手用园艺姿态，佛像用攻击姿态）
4. 场景 vs 角色（童话世界里的黑暗角色）
5. 色彩 vs 情绪（用暖色表达悲伤/冷色表达愤怒）
6. 服装 vs 角色（正装角色在垃圾场，流浪汉在宫殿）
7. 比例 vs 真实（超大头/超小身子，制造不安感）
```

## 第三章：角色一致性维护

> AI视频逐镜头生成，无记忆。只认"参考图+提示词+空间约束"。

### 问题根源

```
镜头1：角色A侧脸 ✅
镜头2：角色A全身 → 模型不认识角色A → 变成另一个人 ❌
```

真正强约束 = **参考图（像素级）+ 提示词（语言级）+ 镜头设计（空间约束）**

### 三层锚定系统

**第一层：参考图锚定（最强约束）**

```
标准配置：角色立绘三件套
1. 正脸图（正面头像/上半身）
2. 侧脸图（90°侧脸）
3. 全身图（站立姿态）
```

**第二层：DNA标签卡锚定（核心）**

```
【角色名】
- 面部几何：[脸型/颧骨/眉形/眼形/鼻形/嘴形/下巴]
- 体型：[身高/肩宽/体型/重心位置]
- 姿态习惯：[标志性姿态/重心/站姿/坐姿]
- 服装：[主色/辅色/标志性单品]
- 色彩锚点：[主色/辅色/点缀色]
- 标志性细节：[疤/痣/眼镜/配件]
```

**第三层：镜头设计锚定（空间约束）**

```
每个镜头必须包含：
□ 角色在画面中的精确位置（左1/3/右1/3/中）
□ 角色与相机的距离（近/中/远/特写）
□ 角色与道具/其他角色的空间关系
□ 视线方向（看哪里）
```

### 一致性检查清单（每镜必查）

```
□ 参考图是否为同角色的同一服装/姿态？
□ 提示词中DNA标签是否完整（面型/体型/服装/细节）？
□ 镜头设计中角色位置是否与前后镜头一致？
□ 色彩锚点（主色/辅色）是否出现在提示词中？
□ 标志性细节（疤/痣/眼镜）是否被提及？
□ 光影方向是否影响角色色彩呈现？
□ 镜头距离是否影响角色细节可见度？
```

### 提示词优化迭代流程

```
Step 1: 初始生成
   → DNA完整描述 + 参考图 + 场景要求

Step 2: 第一轮评估
   → 检查面部/体型/服装/色彩/细节是否一致

Step 3: 问题定位
   → 面部出错？体型出错？服装出错？色彩出错？

Step 4:针对性修复
   → 面部：强化面部几何描述
   → 体型：补充身高肩宽数据
   → 服装：明确主色+辅色+材质
   → 色彩：加入色彩锚点描述

Step 5: 第二轮生成
   → 修复版提示词 + 同参考图

Step 6: 一致性检查
   → 与前后镜头对照

Step 7: 如仍有问题，返回Step 3

Step 8: 定稿输出
   → 最终提示词 + 参考图打包存储
```

### AI平台一致性参数

| 参数 | 即梦 | 海螺 | 可灵 |
|------|------|------|------|
| 参考图数量 | 1-3张 | 1张 | 1-2张 |
| 角色DNA描述 | 必须完整 | 必须完整 | 必须完整 |

## 第四章：表情系统与姿态设计

> 写实角色设计中的表情联动与姿态习惯。

### 情绪-表情-姿态联动

情绪、表情、姿态是三个互相咬合的齿轮。在AI视频生成时，必须让三者同时出现在Prompt中，才能让角色的情绪传达准确。

#### 八种核心情绪联动矩阵

| 情绪 | 眼睛 | 眉毛 | 嘴巴 | 姿态 |
|------|------|------|------|------|
| **愤怒** | 缩小、下压 | +30°到+60°上扬 | 锯齿型/横线抿紧 | 身体前倾、拳头紧握、重心前移 |
| **悲伤** | 瞳孔下垂 | +20°到+40°下压 | 下垂型 | 肩膀内收、身体下沉、重心偏下 |
| **恐惧** | 瞳孔放大、高光消失 | 抬起 | O型微张 | 身体后退、双臂护胸、重心后移 |
| **惊讶** | 瞳孔放大 | -30°到-60°下沉 | O型 | 身体僵住、双手展开、重心居中 |
| **厌恶** | 微眯 | 下压 | 上翘 | 身体后仰、头部后移、偏头 |
| **轻蔑** | 眯眼 | 不对称 | 单边上扬 | 身体后仰、单手插兜、偏头 |
| **喜悦** | 弯月形 | 上扬 | 上扬型 | 身体放松、肩膀抬起、重心居中 |
| **平静** | 正常 | 水平（-5°到+5°） | 横线型 | 身体放松、重心居中、肌肉松弛 |

#### 愤怒三型（最常用）

```
压抑型：嘴角抿成直线，拳头握紧垂在身侧
爆发型：川字眉，牙齿咬紧，身体前倾双拳举起
威压型：眉心无皱纹，嘴角微微下压，身体静止不动
```

### 表情细节参数

#### 眼睛8态参数

| 眼睛状态 | 瞳孔位置 | 眼皮状态 | 高光位置 | 用于表达 |
|----------|----------|----------|----------|----------|
| 正常 | 瞳孔居中 | 正常 | 右上 | 基准状态 |
| 惊讶 | 瞳孔放大 | 眼皮抬起 | 缩小 | 震惊、呆住 |
| 愤怒 | 瞳孔缩小 | 眼皮下压 | 缩小或消失 | 生气、敌意 |
| 恐惧 | 瞳孔放大 | 眼皮完全睁开 | 消失 | 惊恐、惊吓 |
| 得意 | 瞳孔缩小 | 眼皮下压 | 右上 | 嚣张、得意 |
| 悲伤 | 瞳孔略下垂 | 上眼皮上抽 | 缩小 | 委屈、难过 |
| 斗鸡眼 | 向中心聚集 | — | — | 懵圈、困惑 |
| 呆萌 | 瞳孔缩小 | 正常 | 正常 | 平静、呆滞 |

#### 眉毛角度密码

以水平线为0°，测量眉毛与水平线的夹角：

| 情绪 | 角度范围 | 眉毛形态描述 |
|------|----------|-------------|
| 愤怒 | +30°到+60° | 眉尾上扬，眉头下压，呈"倒八"形 |
| 惊讶 | -30°到-60° | 眉尾下沉，眉头上扬，呈"正八"形 |
| 悲伤 | +20°到+40° | 眉头下压，眉尾相对稳定 |
| 平静 | -5°到+5° | 几乎水平，无明显起伏 |

#### 嘴巴6种变化

| 嘴巴形状 | 适用情绪 | 视觉特征 |
|----------|----------|----------|
| O型 | 惊讶、张嘴 | 嘴唇圆形张开 |
| 横线型 | 抿紧、生气 | 嘴唇平直紧绷 |
| 上扬型 | 微笑、得意 | 嘴角向上弯曲 |
| 下垂型 | 沮丧、委屈 | 嘴角向下弯曲 |
| 锯齿型 | 愤怒、大喊 | 嘴唇边缘呈锯齿状 |
| 月牙型 | 傻笑、开心 | 嘴角大弧度上弯 |

### 姿态设计

#### 姿态公式

```
姿态 = 重心位置 × 四肢角度 × 脊椎曲直 × 头部位置
```

#### 三原则

1. **重心**：高重心=自信（直立），低重心=压抑（下沉）
2. **四肢**：张开=开放，紧缩=封闭
3. **脊椎**：直=力量，弯=谦卑/疲惫

#### 姿态习惯设计

每个角色应有独特的姿态习惯：

| 类型 | 示例 |
|------|------|
| 站姿 | 挺拔直立/微微驼背/三七分重心/丁字步 |
| 手部习惯 | 双手插兜/抱胸/背手/自然下垂/叉腰 |
| 坐姿 | 端坐（正式）/瘫坐（放松）/翘二郎腿 |

### 视线与眼神设计

| 视线方向 | 叙事含义 |
|---------|---------|
| 看镜头（看观众） | 直面交流，邀请参与 |
| 看角色A | 关注角色A，观众被引导看A |
| 看画面外左侧 | 画面外有信息，引发好奇 |
| 看画面外右侧 | 回忆/期待 |
| 低头/看地 | 内省、压抑、失败、羞愧 |
| 仰头/看天 | 希望、祈祷、放弃、死亡 |
| 瞳孔高光消失 | 死亡、精神崩溃 |

### 配饰的视觉锚点作用

```
配饰设计要点：
□ 位置精确（左耳/右手/颈部）
□ 形状具体（圆形/方形/十字架）
□ 材质明确（金属/皮革/布料）
□ 色彩独特（与服装形成对比）
□ 尺寸比例（过大/过小/适中）
```

## 第五章：角色关系设计（Blocking）

> AI不理解"谁看谁"，需要用语言精确约束角色间的空间关系。
> Blocking = 角色在空间中的精确位置和朝向。

### 站位关系基础

**横向三分**：
```
[角色A] | [留白/道具] | [角色B]
```
- **角色A左1/3，角色B右1/3**：标准对立关系
- **角色A左1/3，角色B中1/3**：亲密关系（近）
- **角色A中1/3，角色B右1/3**：主次关系（A主B次）

**竖向三分**：
```
[后排角色（虚）]
[前排角色（实）]
```
- **前景清晰、后景虚化**：主次关系，前景是主角
- **前后都清晰**：平等关系

### 视线是叙事工具

| 视线方向 | 叙事含义 |
|---------|---------|
| A看B | A关注B，观众被引导看B |
| A看画面外左侧 | 画面外有信息，观众好奇 |
| A低头/看地 | 内省、压抑、失败、羞愧 |
| A仰头/看天 | 希望、祈祷、放弃、死亡 |
| A斗鸡眼 | 懵圈、喜剧效果 |
| A瞳孔高光消失 | 死亡、精神崩溃 |

**视线优先级**：当多个角色同屏，观众视线顺序 = 角色重要性顺序。

### 六种构图类型

| 类型 | 结构 | 适用 |
|------|------|------|
| **对称构图** | A左B右镜像 | 平等对峙、谈判 |
| **三角形** | A顶点，B左下，C右下 | 三人决策、辩论 |
| **对角线** | A左上B右下 | 张力、对立、冲突 |
| **中心构图** | A居正中 | 强调重要性、孤立感 |
| **三分法** | A右1/3，B左1/3 | 平衡但有重点 |
| **S型曲线** | 多人排序 | 多人排序、节奏感 |

### Blocking六种站位类型

**1. 对峙型（2人）**：
```
A站在左1/3，B站在右1/3，
A面朝右，B面朝左，
两人视线在画面中央交汇。
中间留白 = 张力空间。
```

**2. 导师-学生型（2人）**：
```
导师站在右1/3，学生站在左1/3，
导师面朝左下方（看学生），
学生面朝右上方（仰望导师），
导师比学生高（坐姿 vs 站姿）。
```

**3. 三角站位型（3人）**：
```
角色A：画面左侧1/3，主导位
角色B：画面右侧1/3，辅助位
角色C：站在A和B的后方中间，略高
```

**4. 孤立型（中心单角色）**：
```
主角站在画面正中央，
其他角色站在外围或画面边缘，
主角居中 = 被聚焦 / 被围攻 / 独白
```

**5. 包围型（4人以上）**：
```
四人包围圈：
A在左上，B在右上，C在左下，D在右下，
E或空在中央 = 被包围者
```

**6. 纵深追踪型（2人以上）**：
```
A在前（近景/中景），B在后（远景/全景），
同一方向站立或行走，
镜头在A和B之间切换
```

### Blocking检查清单（每镜必查）

```
□ 每个角色的画面位置是否明确（左/中/右/上/下）？
□ 角色视线方向是否精确描述（看向哪里/看向谁）？
□ 角色间空间关系是否与前后镜头一致？
□ 视线有没有"打架"（两个角色互相看对方）？
□ 越轴是否被处理（角色换了位置后如何过渡）？
□ 光影方向是否锁定（不因镜头切换而变）？
```

## 第六章：负面设计清单

> 按阶段分类：设计阶段 → 提示词阶段 → 生成阶段 → 后期阶段。

### 一、设计阶段错误

**❌ 错误1：只做主角，不做群像**
- 严重等级：🔴 致命
- 后果：观众分不清谁是谁；AI视频多角色镜头全融在一起
- 正确做法：先出群像，再从群像里提取个体

**❌ 错误2：原型单一化**
- 严重等级：🟠 高
- 后果：角色扁平无聊，观众记不住
- 正确做法：每个主要角色至少叠加两个原型

**❌ 错误3：剪影无辨识度**
- 严重等级：🟡 中
- 测试方法：只给AI看剪影，问"能认出是谁吗"？

**❌ 错误4：角色太"完美"**
- 严重等级：🟡 中
- 后果：角色没有记忆点，无法形成文化符号

### 二、提示词阶段错误

**❌ 错误5：提示词里不写角色DNA**
- 严重等级：🔴 致命
- 正确做法：每条提示词必须包含角色DNA标签，至少3项

**❌ 错误6：多人场景不写角色区分**
- 严重等级：🔴 致命
- 正确做法：写明每个角色的色彩/体型/位置差异

**❌ 错误7：不写光影方向**
- 严重等级：🟠 高
- 正确做法：每条提示词必须写光影方向描述

**❌ 错误8：服装色彩太接近**
- 严重等级：🟠 高
- 正确做法：同场景角色色彩差距必须拉到最大

### 三、生成阶段错误

**❌ 错误9：不传参考图/三视图不齐全**
- 严重等级：🔴 致命
- 正确做法：每个角色必须有标准三视图

**❌ 错误10：景别跳跃太大**
- 严重等级：🟠 高
- 正确做法：远景→中景→近景（渐次OK），避免跳跃

**❌ 错误11：运动强度过高**
- 严重等级：🟠 高
- 正确做法：打斗/奔跑选"弱"或"中"

### 四、后期阶段错误

**❌ 错误12：不检查就过**
- 严重等级：🔴 致命
- 正确做法：每个镜头输出后逐项核验

**❌ 错误13：角色标志性细节丢失**
- 严重等级：🟠 高
- 正确做法：每条提示词里必须包含标志性细节

### 快速检查表（每镜必查）

```
【设计阶段】
□ 群像先于个体设计？
□ 每个角色至少2个原型混搭？
□ 剪影有辨识度？

【提示词阶段】
□ DNA标签≥3项？
□ 多人场景写了色彩区分？
□ 光影方向锁定？

【生成阶段】
□ 三视图参考图齐全？
□ 景别跨度合理？
□ 运动强度匹配动作类型？

【后期阶段】
□ 标志性细节都在？
□ 服装主色没变？
□ 脸型一致？

全部通过 → 下一镜
任一失败 → 重新生成
```

## 第八章：中国传统角色设计

### 动态签名模板

**书生**：
- 姿态：身体微微前倾，重心偏后
- 手部：常以袖手姿势为主
- 步态：步幅小，节奏缓慢
- 标志性动作：抚须、执卷、负手而立

**武将**：
- 姿态：身体挺直，肩宽体壮
- 手部：常握拳或按剑
- 步态：步幅大，节奏沉稳
- 标志性动作：拔剑、勒马、托塔

**儿童**：
- 姿态：活泼好动，身体圆润
- 手部：常张开或指物
- 步态：蹦跳，跑动为主
- 标志性动作：嬉戏、哭闹、撒娇

**老人**：
- 姿态：身体微驼，重心下沉
- 手部：常拄杖或背手
- 步态：步幅小，步速缓慢
- 标志性动作：咳嗽、叹气、远眺

### 水墨质感参数

**浓淡五色**：
```
浓墨（焦墨）：100%墨色，轮廓、强调
淡墨：40-60%墨色，层次过渡
清墨：10-20%墨色，远景、雾气
爆墨：浓淡混合，爆炸核心
留白：0%墨色，负空间、呼吸感
```

**墨迹边缘类型**：
| 类型 | 边缘特征 | 适用 |
|------|----------|------|
| 渗透型 | 墨水向纸纤维扩散 | 背景、远景 |
| 飞白型 | 笔触快速划过留白 | 残影、速度线 |
| 积墨型 | 墨水堆积，边缘深浅 | 重量感、撞击点 |
| 泼墨型 | 大面积不可控扩散 | 爆炸、情绪爆发 |

### 参考图哲学

> 给AI一张图，胜过写一万字提示词。

AI 视频/图片模型接收参考图后，会从中"推演"出：
- 角色的**体型与动作规律**
- 角色的**声音气质**
- 角色的**情绪演技风格**
- 角色的**行走姿势与肢体语言**

### 成片即前期：IP宇宙思维

| 旧认知 | 新认知 |
|--------|--------|
| 做剧 = 最终产品 | 做剧 = 世界观前期容器 |
| 写小说 = 写小说 | 写小说 = 为游戏化/动画化做前期 |
| 成片 = 终点 | 成片 = 动态分镜，二创的起点 |
| 观众 = 消费者 | 观众 = 潜在二创者/共建者 |

**互动创作生态的终极闭环**：

```
创作者做前期（世界观/角色设计）
    ↓
成片上线（= 发布可互动的世界观原型）
    ↓
观众截图、做表情包、写同人、二创视频
    ↓
角色IP自然裂变扩散
```
