## 概述

Character Design Skill 是一套完整的AI视频角色设计方法论，涵盖从概念到Prompt的完整流程。适用于**即梦**、**可灵**、**海螺**等AI视频生成平台。

### 设计理念

| 旧范式 | 新范式 |
|--------|--------|
| 让角色讨人喜欢 | 让角色**令人难忘** |
| 帅哥靓女 | 不完美、有缺陷、有争议 |
| 单一原型 | **原型混搭** |
| 近似色、和谐配色 | **色彩暴力** |
| 固定姿势站桩 | **姿态叙事** |

**核心原理**：角色的价值完全取决于他与其他角色的**差异性**，而非自身的完美。

## 核心能力

### 1. 角色DNA七字诀

设计角色的七个维度，全部用"差异性"贯穿：

```
群像 → 原型 → 姿态 → 情绪联动 → 色彩 → 服化道 → 极端对比
```

### 2. 角色一致性维护

解决AI视频逐镜头生成时的角色一致性问题：

- **三层锚定系统**：参考图 + DNA标签 + 镜头设计
- **跨平台参数**：即梦/海螺/可灵各有差异
- **一致性检查清单**：每镜必查

### 3. Blocking视线叙事

精确控制多角色场景中的空间关系：

- 六种构图类型（对称/三角/对角线等）
- 六种站位类型（对峙/导师-学生/三角等）
- 十种视线叙事

### 4. 群像互补设计

角色只有在关系中才有意义：

- 性格/色彩/体型/姿态/原型五维互补
- 群像色彩隔离方案
- 视线优先级设计

### 5. AI提示词模板

可快速套用的Prompt模板：

- 角色立绘Prompt模板
- 镜头Prompt模板
- 群像Prompt模板
- 情绪镜头Prompt模板

### 6. 负面设计清单

16个常见错误，按阶段分类：

- 设计阶段（4个）
- 提示词阶段（4个）
- 生成阶段（3个）
- 后期阶段（5个）

### 02_角色一致性维护

AI视频角色设计的核心难题：

```
镜头1：角色A侧脸 ✅
镜头2：角色A全身 → 模型不认识角色A → 变成另一个人 ❌
```

解决方案：
- 三层锚定系统
- 跨平台一致性参数
- Shot-to-Shot参考链
- 常见失败案例与修复

### 03_角色关系设计Blocking

AI不理解"谁看谁"，需要用语言精确约束：

- 站位关系基础（三分法）
- 视线是叙事工具
- 六种构图类型
- 六种Blocking站位类型
- Blocking检查清单

### 04_表情系统与姿态设计

### 05_负面设计清单

按阶段分类的常见错误：

**设计阶段错误**：
- ❌ 只做主角，不做群像
- ❌ 原型单一化
- ❌ 剪影无辨识度
- ❌ 角色太"完美"

**提示词阶段错误**：
- ❌ 不写角色DNA
- ❌ 多人场景不写角色区分
- ❌ 不写光影方向
- ❌ 服装色彩太接近

**生成阶段错误**：
- ❌ 不传参考图
- ❌ 景别跳跃太大
- ❌ 运动强度过高

**后期阶段错误**：
- ❌ 不检查就过
- ❌ 角色标志性细节丢失
