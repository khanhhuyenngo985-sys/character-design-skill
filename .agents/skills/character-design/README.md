## 核心能力

- **角色DNA七字诀**：群像→原型→姿态→情绪联动→色彩→服化道→极端对比
- **角色一致性维护**：三层锚定系统 + 跨平台参数（即梦/海螺/可灵）
- **Blocking视线叙事**：六种构图类型 + 六种站位类型
- **群像互补设计**：性格/色彩/体型/姿态/原型五维互补
- **AI提示词模板**：角色立绘/镜头/群像/情绪Prompt模板
- **负面设计清单**：设计/提示词/生成/后期四个阶段的常见错误
