# 中国传统角色设计

---

## 传统角色类型

### 书生

**外貌特征**：
- 脸型：椭圆、脸庞清秀
- 眉形：清秀、眉峰不明显
- 眼形：细长、眼神温和
- 发型：束发戴巾、高髻
- 体型：清瘦、肩窄

**服饰特征**：
- 交领长袍（宽袖/窄袖）
- 腰带系于胸部下方
- 布鞋或翘头履
- 颜色：青/白/淡蓝/淡绿

**动态签名**：
| 要素 | 描述 |
|------|------|
| 姿态 | 身体微微前倾，重心偏后 |
| 手部 | 常以袖手姿势为主 |
| 步态 | 步幅小，节奏缓慢 |
| 标志性动作 | 抚须、执卷、负手而立、踱步思考 |

**性格特征**：
- 文弱但有骨气
- 谦逊但不失傲骨
- 内心坚定外表温和

**AI Prompt模板**：
```
书生，20-30岁，面容清秀，
眉形清秀/眼形细长眼神温和，
发型束发戴巾/高髻，
体型清瘦/肩窄/身高170-175cm，
服饰：青色交领长袍/宽袖/白色腰带/黑色布鞋，
色彩锚点：青/白/黑，
姿态：微微前倾/重心偏后/袖手，
表情：眉头舒展/嘴角微扬，
光源：自然光/左上方45°侧光，
背景：[场景描述]
水墨风格，16:9
```

### 武将

**外貌特征**：
- 脸型：方正、棱角分明
- 眉形：粗黑、眉峰明显
- 眼形：大眼、眼神刚毅
- 发型：髡发或束发
- 体型：魁梧、肩宽体壮

**服饰特征**：
- 铠甲（鳞甲/锁子甲/板甲）
- 战靴（高帮/厚底）
- 披风（常见红/黑）
- 护腕、护肩

**动态签名**：
| 要素 | 描述 |
|------|------|
| 姿态 | 身体挺直，肩宽体壮 |
| 手部 | 常握拳或按剑 |
| 步态 | 步幅大，节奏沉稳 |
| 标志性动作 | 拔剑、勒马、托塔、横槊、扬鞭 |

**性格特征**：
- 豪迈奔放
- 勇武但不一定有谋
- 忠诚或背叛

**AI Prompt模板**：
```
武将，30-40岁，面容刚毅，
眉形粗黑眉峰明显/眼形大眼眼神刚毅，
发型髡发/束发，
体型魁梧/肩宽/身高180-185cm/体格健壮，
服饰：青铜鳞甲/红色披风/黑色战靴/护肩/护腕，
色彩锚点：青铜/红/黑，
姿态：身体挺直/重心居中/双手自然下垂，
表情：眉头紧锁/嘴角紧闭/目光如炬，
光源：上方或侧上方强光，
背景：[战场/军营/宫殿]
水墨风格，16:9
```

### 儿童

**外貌特征**：
- 脸型：圆润、脸颊丰满
- 眉形：淡、眉峰未显
- 眼形：大而圆、眼神清澈
- 发型：总角（两边束发）或垂髫
- 体型：矮小、四肢短小

**服饰特征**：
- 肚兜、束腰
- 短衫短裤或背带裤
- 颜色：红/黄/绿等明亮色

**动态签名**：
| 要素 | 描述 |
|------|------|
| 姿态 | 活泼好动，身体圆润 |
| 手部 | 常张开或指物 |
| 步态 | 蹦跳，跑动为主 |
| 标志性动作 | 嬉戏、哭闹、撒娇、躲藏、骑竹马 |

**性格特征**：
- 天真无邪
- 好奇活泼
- 或懂事乖巧或顽皮捣蛋

**AI Prompt模板**：
```
儿童，6-10岁，面容圆润可爱，
眉形淡眉毛未显/眼形大而圆眼神清澈，
发型总角或垂髻，
体型矮小/四肢短小/身高100-130cm，
服饰：红色肚兜/黄色短裤/蓝色背带/布鞋，
色彩锚点：红/黄/蓝/绿，
姿态：身体活泼/重心不稳/动作多变，
表情：笑容灿烂/或哭闹/或好奇，
光源：柔和自然光，
背景：[庭院/巷子/市集]
水墨风格，16:9
```

### 老人

**外貌特征**：
- 脸型：瘦削、皱纹
- 眉形：长眉、眉尾下垂
- 眼形：眼角下垂、眼神深邃
- 发型：白发、秃顶或稀疏
- 体型：驼背、消瘦

**服饰特征**：
- 长袍（深色）
- 拐杖（常见）
- 帽子（斗笠/员帽）
- 颜色：灰/褐/墨

**动态签名**：
| 要素 | 描述 |
|------|------|
| 姿态 | 身体微驼，重心下沉 |
| 手部 | 常拄杖或背手 |
| 步态 | 步幅小，步速缓慢 |
| 标志性动作 | 咳嗽、叹气、远眺、捻须、垂钓、棋局 |

**性格特征**：
- 历经沧桑
- 智慧或狡黠
- 或慈祥或阴沉

**AI Prompt模板**：
```
老人，60-80岁，面容沧桑，
眉形长眉眉尾下垂/眼形眼角下垂眼神深邃，
发型白发/秃顶/稀疏，
体型驼背/消瘦/身高160-165cm，
服饰：灰色长袍/褐色外套/黑色布鞋/拐杖，
色彩锚点：灰/褐/墨/白，
姿态：身体微驼/重心下沉/拄杖或背手，
表情：眉头舒展/嘴角下撇/目光深远，
光源：自然光/暖色调，
背景：[山间/江边/庭院/亭台]
水墨风格，16:9
```

### 女性角色（古装）

**外貌特征**：
- 脸型：瓜子脸、鹅蛋脸
- 眉形：柳叶眉、远山眉
- 眼形：凤眼、丹凤眼
- 发型：高髻、鬟、辫
- 体型：纤细、柔美

**服饰特征**：
- 襦裙（齐腰/齐胸）
- 对襟/交领
- 披帛、云肩
- 绣花鞋

**动态签名**：
| 要素 | 描述 |
|------|------|
| 姿态 | 身体挺直，肩部稍窄 |
| 手部 | 常以帕掩口或抚发 |
| 步态 | 步幅小而快，碎步 |
| 标志性动作 | 拂袖、回眸、拈花、理妆、绣花、抚琴 |

**性格特征**：
- 温柔婉约或刚烈果敢
- 多才多艺
- 或贤淑或刁蛮

**AI Prompt模板**：
```
古代女子，18-30岁，面容秀美，
眉形柳叶眉/眼形凤眼眼神含情，
发型高髻/鬓角垂发/步摇/珠钗，
体型纤细/柔美/肩窄/身高160-165cm，
服饰：淡青色齐腰襦裙/白色披帛/绣花鞋，
色彩锚点：青/白/粉/金，
姿态：身体挺直/肩部稍窄/双手交叠于身前，
表情：眉目含情/嘴角微扬/目光流转，
光源：柔和侧光/暖色调，
背景：[闺房/庭院/楼阁/溪边]
水墨风格，16:9
```

## 水墨质感参数

### 浓淡五色

| 墨色 | 浓度 | 用途 |
|------|------|------|
| 焦墨（焦浓） | 100% | 轮廓、强调、动作线 |
| 浓墨 | 80% | 近景、重要细节 |
| 重墨 | 60% | 中景、标准描绘 |
| 淡墨 | 40-60% | 层次过渡、中景 |
| 清墨（浅墨） | 10-20% | 远景、雾气、背景 |
| 留白 | 0% | 负空间、呼吸感 |

### 墨迹边缘类型

| 类型 | 边缘特征 | 适用 |
|------|----------|------|
| 渗透型 | 墨水向纸纤维扩散 | 背景、远景、慢动作残影 |
| 飞白型 | 笔触快速划过留白 | 残影、速度线、快速运动 |
| 积墨型 | 墨水堆积，边缘深浅 | 重量感、撞击点、挤压部位 |
| 泼墨型 | 大面积不可控扩散 | 爆炸、情绪爆发、极限动作 |
| 破墨型 | 浓淡相破 | 烟雾、云水、过渡效果 |

### 笔法与角色动态

| 笔法 | 特征 | 适用角色/动作 |
|------|------|----------------|
| 高古游丝描 | 细线均匀，粗细一致 | 书生、仕女、慢动作 |
| 铁线描 | 粗细均匀，凝重有力 | 武将、沉稳动作 |
| 兰叶描 | 线描如兰叶，变化丰富 | 飘带、衣物、动态 |
| 减笔描 | 简练概括，寥寥数笔 | 写意、远景、次要人物 |
| 混描 | 浓淡并用 | 体积感、立体感 |
| 泼墨法 | 大笔挥洒 | 背景、情绪、远景 |

## 水墨动画技法

### 留白动态

```
空白 = 观众想象空间
留白比例：40%-60%为最佳

留白类型：
1. 人物周围留白（呼吸感）
2. 画面边缘留白（延伸感）
3. 人物内心留白（情绪张力）
```

### 散点透视

```
中国画视角：多焦点同时存在
优点：不受单一视角限制
适用：场景展示、角色介绍、叙事全景

西方透视：单焦点
优点：真实感强
适用：对话、特写、焦点场景
```

### 动态表达

| 笔法 | 适用 | 效果 |
|------|------|------|
| 兰叶描 | 衣纹飘动，线条流畅 | 优雅、流动 |
| 铁线描 | 静止部分，线条稳定 | 沉稳、有力 |
| 减笔描 | 动作激烈，简洁有力 | 速度、力量 |
| 混描 | 体积感强 | 写实、厚重 |

## 传统角色配色方案

### 书生配色

```
主色：青/蓝 → 象征清雅、高洁
辅色：白/灰 → 象征纯净、素雅
点缀：墨/褐 → 象征书卷气

推荐色板：
- 青衫书生：青/白/墨
- 白衣书生：白/淡蓝/淡灰
- 褐袍书生：褐/青/白
```

### 武将配色

```
主色：黑/灰/铜 → 象征铠甲、战场
辅色：红/金 → 象征热血、荣耀
点缀：白/蓝 → 象征旗帜、天空

推荐色板：
- 黑甲武将：黑/红/金
- 银甲武将：银白/蓝/红
- 布衣武将：褐/青/黑
```

### 女性配色

```
主色：粉/红 → 象征娇美、柔媚
辅色：青/白 → 象征清丽、纯洁
点缀：金/紫 → 象征富贵、神秘

推荐色板：
- 闺秀：粉/白/金
- 侠女：青/黑/红
- 宫妃：紫/红/金
```
