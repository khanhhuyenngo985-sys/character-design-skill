# 角色关系设计（Blocking）

> AI不理解"谁看谁"，需要用语言精确约束角色间的空间关系。
> Blocking = 角色在空间中的精确位置和朝向。AI视频里Blocking = "谁在哪、谁看哪"的语言约束。

---

## 站位关系基础

### 横向三分

```
[角色A] | [留白/道具] | [角色B]
```

- **角色A左1/3，角色B右1/3**：标准对立关系
- **角色A左1/3，角色B中1/3**：亲密关系（近）
- **角色A中1/3，角色B右1/3**：主次关系（A主B次）
- **角色A中1/3，角色B中1/3**：重叠（需要明确谁在前）

### 竖向三分

```
[后排角色（虚/后景）]
[中排角色（中景）]
[前排角色（实/前景）]
```

- **前景清晰、后景虚化**：主次关系，前景是主角
- **前后都清晰**：平等关系
- **前虚后实**：强调后景角色

### 景深关系

| 前后关系 | 清晰度 | 叙事含义 |
|---------|--------|---------|
| 前清后虚 | 前景实 | 主角在前景，紧张/压力 |
| 前虚后清 | 后景实 | 主角在后景，安全/旁观 |
| 前后都清 | 同等清晰 | 平等关系/群像 |

## 视线是叙事工具

### 视线基础方向

| 视线方向 | 叙事含义 | AI Prompt写法 |
|---------|---------|--------------|
| A看B | A关注B，观众被引导看B | "A looking at B" |
| A看画面外左侧 | 画面外有信息，观众好奇 | "A looking off-screen left" |
| A看画面外右侧 | 画面外有信息，观众好奇 | "A looking off-screen right" |
| A低头/看地 | 内省、压抑、失败、羞愧 | "A looking down" |
| A仰头/看天 | 希望、祈祷、放弃、死亡 | "A looking up" |
| A斗鸡眼 | 懵圈、喜剧效果 | "A cross-eyed" |
| A瞳孔高光消失 | 死亡、精神崩溃 | "A eyes losing highlight" |
| A瞳孔高光收缩 | 集中、警惕、紧张 | "A pupils constricting" |
| A眼睛微闭 | 疲惫、满足、蔑视 | "A eyes half-closed" |

### 视线优先级法则

**核心原则**：当多个角色同屏，观众视线顺序 = 角色重要性顺序。

**视线打架**：当两个角色的视线互相看向对方，观众会在两人之间切换注意力。

**视线汇聚**：当多个角色同时看向一点，该点成为画面焦点。

### 视线越轴（Breaking Axis）

**问题**：A和B正在对话（A在左看右，B在右看左）。突然C从A的后面走到B的位置。

**错误**：
```
镜头1：A在左看右，B在右看左（建立关系）
镜头2：C突然出现在B的位置（越轴，观众困惑）
```

**正确**：
```
方法1：C先走出画面 → 重新建立新的空间关系
方法2：C先看向A，建立关系 → 再转向B
方法3：用镜头剪切过渡，避免直接越轴
```

## 六种构图类型

### 1. 对称构图（Symmetrical）

| 项目 | 内容 |
|------|------|
| 结构 | A左B右镜像 |
| Prompt关键词 | "A在左B在右，左右对称" |
| 适用 | 平等对峙、谈判、镜像关系 |
| 不适用 | 力量悬殊、上下级、亲密关系 |

**案例**：
- 谈判桌上两人僵持
- 双胞胎对决
- 正邪对立

### 2. 三角形（Triangular）

| 项目 | 内容 |
|------|------|
| 结构 | A顶点，B左下，C右下 |
| Prompt关键词 | "A在顶点，B左C右" |
| 适用 | 三人决策、辩论赛、ABC关系 |
| 不适用 | 平等友情三人组（应用直线排列） |

**变体**：
- 倒三角：A在下，B左上，C右上（弱势位置）
- 侧三角：三点在一侧（排队/列阵）

### 3. 对角线（Diagonal）

| 项目 | 内容 |
|------|------|
| 结构 | A左上B右下 |
| Prompt关键词 | "A在左上B在右下" |
| 适用 | 张力、对立、冲突、动态关系 |
| 不适用 | 亲密场景、静态关系 |

**对角线变体**：
- 人物分布在对角线上
- 强调距离和张力
- 视线沿对角线交汇

### 4. 中心构图（Centered）

| 项目 | 内容 |
|------|------|
| 结构 | A居正中 |
| Prompt关键词 | "A站在画面正中央" |
| 适用 | 强调重要性、孤立感、聚焦 |
| 不适用 | 需要展示关系网、展示多角色互动 |

**中心构图的变体**：
- 主角居中配角色围观（被围攻/审判）
- 主角居中配留白（孤立/独白）
- 主角居中配角散在外围（众星捧月/团队展示）

### 5. 三分法（Rule of Thirds）

| 项目 | 内容 |
|------|------|
| 结构 | A右1/3，B左1/3 |
| Prompt关键词 | "A在右侧1/3，B在左侧1/3" |
| 适用 | 平衡但有重点、主次分明 |
| 不适用 | 两人平等对峙（无留白张力） |

**三分法的优势**：
- 画面有留白，呼吸感
- 主角在黄金分割点
- 配角在次要位置

### 6. S型曲线（S-Curve）

| 项目 | 内容 |
|------|------|
| 结构 | 多人沿S形排列 |
| Prompt关键词 | "从左到右A-B-C-D排列成S形" |
| 适用 | 多人排序、节奏感、纵深感 |
| 不适用 | 紧急场景、对峙场景 |

## 六种Blocking站位类型

### 1. 对峙型（2人）

```
A站在左1/3，B站在右1/3，
A面朝右，B面朝左，
两人视线在画面中央交汇。
中间留白 = 张力空间。
→ 用于：谈判、对峙、对话
```

**✅ 完整Prompt（即梦）：**
```
中景镜头，古风书房，
书生站在画面左侧1/3处，面朝右侧，
视线看武将，面部受右上方侧光（亮侧朝书生），
姿态微微前倾，双手垂于身侧。

武将站在画面右侧1/3处，面朝左侧，
视线看书生，处于阴影中（暗侧朝武将），
姿态身体僵直，右手握拳置于腰侧。

两人之间一个半身位留白 = 对峙张力空间。
整体色调冷暖对比（水墨淡彩），
即梦，水墨风格，16:9
```

**适用场景**：
- 谈判桌上两人僵持
- 咖啡馆约会初次见面
- 雨夜巷子对峙
- 决斗前双方对峙

**不适用**：
- 两人力量悬殊/上下级时（用导师-学生型）
- 需要展示亲密关系时
- 两人体型太相似时（留白无张力）

---

### 2. 导师-学生型（2人）

```
导师站在右1/3，学生站在左1/3，
导师面朝左下方（看学生），
学生面朝右上方（仰望导师），
导师比学生高（坐姿 vs 站姿，或站高处 vs 站低处）。
→ 用于：教导、传授、指引场景
```

**✅ 完整Prompt（即梦）：**
```
近景镜头，山崖边，傍晚，
导师（老者）：站在画面右侧1/3处，面朝左下方，
站在高处（石阶上），灰袍大袖，体型清瘦，
视线俯视学生，面部受上方柔光，
姿态：单手抚须，另一手背在身后。

学生（少年）：站在画面左侧1/3处，面朝右上方，
站在低处（草地），粗布短褐，体型瘦小，
视线仰视导师，傍晚侧光打在脸上，
姿态：双手微握，略微低头（敬畏）。

导师比学生高出一个头部的高度，
光影：上方暖光（导师）vs 侧面冷光（学生）。
水墨风格，暖色傍晚光，16:9
```

**适用场景**：
- 师父传授武功
- 领导指点新人
- 父亲教导子女
- 老师教授知识

**不适用**：
- 两人势均力敌/平等对抗时
- 平等友情/爱情时
- 两人已是平辈/同事时

---

### 3. 三角站位型（3人）

```
角色A：画面左侧1/3，主导位
角色B：画面右侧1/3，辅助位
角色C：站在A和B的后方中间，略高
→ 用于：三人决策、辩论赛一方陈述
```

**Prompt模板**：
```
角色A站在画面左侧1/3，面朝B，
角色B站在画面右侧1/3，面朝A，
角色C站在A和B的后方中间位置，略高于两人，
三人形成三角站位。
```

**适用场景**：
- 三人决策会议
- 辩论赛一方陈述
- 夫妻吵架第三方调解
- 首领+副手+护卫

**不适用**：
- 三人平等友情时（可用直线排列）
- 需要展示两方对立+第三方观望时

---

### 4. 孤立型（中心单角色）

```
主角站在画面正中央，
其他角色站在外围或画面边缘，
主角居中 = 被聚焦 / 被围攻 / 独白
→ 用于：主角独自面对压力、独白思考
```

**Prompt模板**：
```
角色A站在画面正中央，面朝镜头，
其他角色（B/C/D）散落在画面边缘/角落，
处于A的外围，A居中是视觉焦点。
```

**适用场景**：
- 主角独自面对压力
- 主角独白思考
- 主角回忆某个画面外的人
- 被审判/被围攻

**不适用**：
- 需要展示角色间紧密互动时
- 群像戏份均衡时

---

### 5. 包围型（4人以上）

```
四人包围圈：
A在左上，B在右上，C在左下，D在右下，
E或空在中央 = 被包围者
→ 用于：四人包围圈、圆桌会议、帮派谈判
```

**Prompt模板**：
```
角色A在左上角，角色B在右上角，
角色C在左下角，角色D在右下角，
四人形成包围圈，
中央E是被包围者（可空出表示物）。
```

**适用场景**：
- 四人包围圈
- 圆桌会议
- 四人帮派谈判
- 篝火晚会
-审讯场景

**不适用**：
- 展示两人亲密关系时
- 需要清晰展示多组关系时

---

### 6. 纵深追踪型（2人以上）

```
A在前（近景/中景），B在后（远景/全景），
同一方向站立或行走，
镜头在A和B之间切换
→ 用于：主角被追、山路纵马追杀、走廊追逐、背后有人跟踪
```

**Prompt模板**：
```
角色A在画面前景（中景/近景），
角色B在画面后景（全景/远景），
两人同向站立/行走，
镜头在A和B之间切换。
```

**适用场景**：
- 主角被追
- 山路纵马追杀
- 走廊追逐
- 背后有人跟踪
- 师徒同行

**不适用**：
- 需要展示两人对话/互动时
- 静止对峙场景

## 视线高阶叙事（10种）

| 序号 | 用法 | Prompt写法 |
|------|------|-----------|
| 1 | 撒谎 | "A看向B，但B的眼睛看向别处（越过我）" |
| 2 | 越轴/打破关系 | "A看向B，此时B转向C，A的视线被切断" |
| 3 | 锁定目标 | "A持枪瞄准镜头，B在A身后准备行动" |
| 4 | 谎言被戳穿 | "A看向B撒谎，B的眼睛突然闭上（A知道被看穿）" |
| 5 | 追随视线 | "观众跟随A的视线看向B，揭示B的秘密" |
| 6 | 视线作为陷阱 | "A向B使眼色，B眨眼回应，C完全没注意到" |
| 7 | 视线回避 | "A在说话但眼睛不看B，B的眼睛一直盯着A" |
| 8 | 视线控制权力 | "A站着说话，B坐着头也不抬，A被迫弯腰" |
| 9 | 视线越轴 | "A和B之间突然出现C，A和B的视线都被C切断" |
| 10 | 视线与物体 | "A看着桌上的枪，所有人的视线都跟着那把枪" |

## Blocking反面案例

### ❌ 反面1：视线混乱

```
❌ Prompt："三个人在聊天"
```

**失败表现**：三个人各看各的方向（有的看镜头，有的看地板，有的看天花板），观众完全不知道谁在跟谁说话。

**原因**：没写视线方向，AI随机分配。

**✅ 正确写法**：
```
角色A站在左侧，面朝右侧，看向角色B
角色B站在中间，面朝左侧，看向角色A
角色C站在右侧，面朝前方，看向画面外左侧（好奇）
```

---

### ❌ 反面2：站位重复

```
❌ Prompt："三个人站在一起合影"
```

**失败表现**：三个人并排站成一排，脸型/体型/服装色彩相近，观众分不清谁是谁。

**原因**：没有用三分法分配位置，三人没有高矮胖瘦差异。

**✅ 正确写法**：
```
角色A（高/壮）站在画面左侧1/3处
角色B（矮/瘦）站在画面右侧1/3处
角色C（中身材）站在中景后方的台阶上
三人形成三角形站位
```

---

### ❌ 反面3：光影不锁定

```
❌ Prompt："角色A在左，角色B在右，两人对话，室内场景"
```

**失败表现**：镜头1角色A脸部右侧受光；镜头2同一角色脸部左侧受光。观众感觉是两个人。

**原因**：没写光影方向，AI每个镜头随机分配。

**✅ 正确写法**：
```
角色A站在左1/3处，面朝右侧，
右上方45°侧光（稳定的光源方向）
角色B站在右1/3处，面朝左侧，
与A相同的光源方向（保持光影一致性）
```

## Blocking检查清单（每镜必查）

### 位置与空间

```
□ 每个角色的画面位置是否明确（左/中/右/上/下/前/后）？
□ 角色间空间关系是否与前后镜头一致？
□ 角色间的距离是否明确（半身位/一个头高/同一位）？
□ 越轴是否被处理（角色换了位置后如何过渡）？
□ 景别变化时Blocking是否需要调整？
```

### 视线与表情

```
□ 角色视线方向是否精确描述（看向哪里/看向谁）？
□ 视线有没有"打架"（两个角色互相看对方）？
□ 适用场景是否匹配当前镜头类型（对峙/亲密/主次）？
□ 视线是否有汇聚点（多点视线交汇处）？
```

### 光影与色彩

```
□ 光影方向是否锁定（不因镜头切换而变）？
□ 同场景角色色彩是否有足够差异？
□ 前清后虚的关系是否符合叙事需求？
```

### 构图与平衡

```
□ 画面重心是否平衡（视觉重量分布）？
□ 适用/不适用是否匹配当前场景类型？
□ 前景/后景的角色是否被区分清楚？
□ 留白是否服务于情绪/张力？
```

## 特殊Blocking场景

### 多人会议场景

```
Prompt：
会议桌横置，
决策者A坐在画面左侧1/3处，
执行者B坐在画面右侧1/3处，
其他角色C/D/E坐在后景/侧景，
A与B形成对立关系，
C/D/E形成围观/辅助关系。
```

### 追逐场景

```
Prompt：
角色A在画面前景（全/中景）奔跑，
角色B在后景跟随（全景/远景），
两人同向运动，
镜头在A和B之间切换。
```

### 对话-反打镜头

```
镜头1（角色A视角）：
A站在左1/3处，面朝右侧，看向B
B站在右1/3处，处于画面中央

镜头2（角色B视角）：
B站在右1/3处，面朝左侧，看向A
A站在左1/3处，处于画面中央

保持：光源方向、角色位置关系、服装色彩
```

## AI视频Blocking注意事项

### 景别跳跃时的Blocking调整

| 跳跃类型 | 是否需要调整Blocking | 说明 |
|---------|---------------------|------|
| 远景→全景 | 不需要 | 保持位置关系 |
| 全景→中景 | 小调整 | 确保中景能容纳全部角色 |
| 中景→近景 | 大调整 | 只能聚焦1-2人 |
| 近景→特写 | 大调整 | 只看1人，其他出画 |
| 特写→近景 | 大调整 | 需要重新建立关系 |

### 运动镜头Blocking

```
□ 角色运动方向是否明确（前/后/左/右）？
□ 摄影机运动是否影响角色关系（推/拉/摇/移）？
□ 运动中视线是否保持（跟拍/盯拍）？
```
