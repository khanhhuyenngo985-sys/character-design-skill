## 色彩稳定性检查清单

```
□ 角色主色饱和度 ≥ 背景饱和度？
□ 角色色彩和环境色彩有足够HSV色相差？
□ 有光源方向辅助主体/背景边缘分离？
□ 肤色角色避开了灰色/白色/浅色背景？
□ 有"色彩锚点"（高饱和局部细节）？
□ 雾/雨/夜间场景有强局部光源？
□ 色彩对比用在了叙事转折点？
```

## 镜头设计锚定检查

```
每个镜头必须包含：
□ 角色在画面中的精确位置（左1/3/右1/3/中/前/后）
□ 角色与相机的距离（远/全/中/近/特）
□ 角色与道具/其他角色的空间关系
□ 视线方向（看向哪里/看向谁）
□ 光源方向（45°侧光/顶光/底光/背光）
□ 景别（决定哪些细节可见）
```

## Prompt优化迭代流程

```
Step 1: 初始生成
   → DNA完整描述 + 参考图 + 场景要求

Step 2: 第一轮评估
   → 检查面部/体型/服装/色彩/细节是否一致

Step 3: 问题定位
   → 面部出错？体型出错？服装出错？色彩出错？

Step 4:针对性修复
   → 面部：强化面部几何描述
   → 体型：补充身高肩宽数据
   → 服装：明确主色+辅色+材质
   → 色彩：加入色彩锚点描述

Step 5: 第二轮生成
   → 修复版提示词 + 同参考图

Step 6: 一致性检查
   → 与前后镜头对照

Step 7: 如仍有问题，返回Step 3

Step 8: 定稿输出
   → 最终提示词 + 参考图打包存储
```

## 负面提示词（避免生成什么）

### 通用负面提示词

```
避免生成：
□ 模糊/低清
□ 畸形/比例失调
□ 色彩失真/偏色
□ 姿态扭曲/不自然
□ 面部特征改变
□ 服装颜色变化
□ 背景杂乱/干扰主体
```

### 场景特定负面提示词

**夜间场景**：
```
避免：
- 面部过暗/看不清
- 霓虹光过曝
- 边缘模糊
- 噪点过多

解决：
- 使用局部光源打脸
- 使用反光材质
- 降低环境亮度
```

**自然场景**：
```
避免：
- 角色融入背景（肤色≈环境色）
- 植物遮挡主体
- 光影混乱

解决：
- 服装色彩与环境对比
- 强光源轮廓分离
- 角色站在空旷区域
```

**多人场景**：
```
避免：
- 角色融在一起
- 视线混乱
- 位置关系不清

解决：
- 使用位置标签（左/中/右）
- 使用色彩隔离
- 使用视线描述
```

## 常见错误修正

### 错误1：角色脸变了

**问题**：每次生成同一角色的脸都不一样

**原因**：没有DNA锚定或参考图不正确

**修正**：
```
✅ "角色A，方脸/颧骨高/眉形粗黑/眼形狭长/鼻梁挺直/下巴方正"
✅ 传正面照作为参考图
```

---

### 错误2：服装颜色变了

**问题**：上一镜红色外套，下一镜变成蓝色

**原因**：没有写明色彩锚点

**修正**：
```
✅ "角色A，红色皮外套（主色）/黑色裤子/黄色围巾（点缀色）"
✅ "保持角色A的红色皮外套不变"
```

---

### 错误3：角色融入背景

**问题**：角色颜色和背景太接近，分不清

**原因**：没有色彩对比设计

**修正**：
```
✅ 角色穿高饱和色彩（红/橙/黄）
✅ 背景设计为低饱和（灰/褐/蓝）
✅ 加入光源分离主体和背景
```
