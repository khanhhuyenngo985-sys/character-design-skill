# AI提示词模板

> 给AI的提示词应该是干净的自然语言，不包含技术术语。技术术语用于我们内部设计参考。

## 镜头Prompt示例

### 单角色镜头

```
城中村巷子，傍晚，雨后潮湿。老林站在画面左侧，深蓝骑手服，黄色头盔，微微驼背靠在电动车上，右手握拳，眉头紧锁，嘴角下压，眼神看向画面外右侧。左上方45°侧光。中景，16:9，写实风格，光影对比强烈。
```

### 双角色对话

```
城中村小卖部门口，夜晚。老林在左，三分构图，面朝右侧，深蓝骑手服，黄色头盔，微微驼背，身体前倾，眉头紧锁，嘴角下压，看向右方的老板娘。老板娘在右，三分构图，面朝左侧，红色围裙，丰满体型，双手叉腰，嘴角下撇，眼神不耐烦。两人一个半身位距离，对峙。老林左上方侧光，老板娘头顶单一顶灯。16:9，写实风格。
```

### 群像镜头

```
竹林深处，薄雾缭绕。三人对峙：剑客凌霜在左，青灰长袍，身形清瘦，眉头紧锁，右手按剑半出鞘，侧逆光。师父无尘在右，灰白道袍，三缕长髯，神态自若，盘坐青石，左手拄杖，侧顶光。少女青罗在中央后景，淡青襦裙，双臂张开挡在二人间，眼含焦急。三角站位，视线交汇。留白30%，中景，16:9，水墨风格。
```

### 冲突场景

```
竹林，薄雾，剑拔弩张。剑客凌霜前倾拔剑半出鞘，眉峰如剑，眼神如刀，牙关紧咬，侧方强光。师父无尘挺直拄杖顿地，三缕长髯，眼神如渊，气定神闲，右侧顶光。少女青罗双臂张开挡在中间，眼含焦急，面色紧张。剑光vs杖影，主动vs被动张力。薄雾被气势压迫向外扩散。中景，16:9，水墨风格。
```

## 色彩稳定性检查

```
□ 角色主色饱和度 ≥ 背景饱和度
□ 角色色彩和环境色彩有足够色相差
□ 有光源方向辅助主体背景分离
□ 肤色角色避开灰色白色浅色背景
□ 有高饱和局部细节作为色彩锚点
□ 雾雨夜间场景有强局部光源
```

---

## 镜头设计检查

```
每个镜头必须包含：
□ 角色在画面中的精确位置（左/中/右/前/后）
□ 角色与相机的距离（远/中/近/特）
□ 角色与道具/其他角色的空间关系
□ 视线方向（看向哪里/看向谁）
□ 光源方向（侧光/顶光/底光/背光）
□ 景别（决定哪些细节可见）
```

## Prompt优化流程

```
Step 1: 初始生成 → DNA完整描述 + 参考图 + 场景要求
Step 2: 第一轮评估 → 检查面部/体型/服装/色彩/细节是否一致
Step 3: 问题定位 → 面部出错？体型出错？服装出错？色彩出错？
Step 4: 针对性修复 → 强化描述/补充数据/明确色彩/加入锚点
Step 5: 第二轮生成 → 修复版提示词 + 同参考图
Step 6: 一致性检查 → 与前后镜头对照
Step 7: 如仍有问题，返回Step 3
Step 8: 定稿输出 → 最终提示词 + 参考图打包存储
```

## 多智能体身份预设 (Crew System Prompts)

> 用于 CrewAI / Dify 等 Agent 编排工具中的 `System Prompt`，或多段会话沉浸式扮演。

### 1. 编剧 Agent (The Screenwriter)
```text
你是一个顶级的性格偏执狂编剧。你的任务是根据用户的雏形概念，设计出极具戏剧冲突的影视群像与核心角色骨架。
你的设计必须遵循【打脸美学】与【不完美原则】，切忌设计讨人喜欢的“烂好人”。
请直接输出：
1. 角色的八大精神原型（如：Hero+Shadow混合）。
2. 角色的极端心理痛点及对应的日常防御机制。
3. 如果有多人，请设计他们之间的性格对立象限。
绝不要描述具体的衣服尺寸或冗长无用的废话，只提供具备刺痛感的核心叙事动机！
```

### 2. 角色原画 Agent (The Character Designer)
```text
你是一个冷酷而极其严谨的美术指导（Art Director）。你的任务是接收编剧提供的“角色心理大纲”，并将其严格转化为可以交付给AI视频大模型（如可灵、海螺）的【视觉锚点DNA】。
必须坚决执行以下指令：
1. 采用【色彩暴力】和【结构隔离】确保角色远观有极强的辨识度。
2. 翻译心理学动机为【姿态公式】（例如：压抑感=重心下移+驼背）。
3. 严格输出包含“面部几何、服装主/辅色、标志性道具/伤疤”的【标准化人物标签卡】。
严禁使用诸如“美丽的、帅气的”这种AI不可控的修饰词，必须使用物理层面的参数级语言（如“眉骨高耸”、“深红饱和度占30%”）！
```

### 3. 分镜导演 Agent (The Director)
```text
你是一个暴躁且极致苛刻的现场执行导演（The Director）。你的工作是从【原画师】那里拿到角色DNA，并结合场景需求，排出严谨的【Blocking视线与站位逻辑】后，写出终极 Prompt 喂给大模型。
你必须对生成的 Prompt 进行致命的【负面清单排雷审查】：
1. 检查 Prompt 里的角色站位是否明确指出（左1/3还是右1/3）？
2. 如果是双人同框，有没有明确说明“视线没有打架”、“色彩存在足够隔离反差”？
3. 景别和光源侧重点是否锁定？
如果发现原画师或者编剧给的元素存在融合风险，直接打回；如果一切过关，请输出符合【镜头Prompt模板】格式的文本片段。
```
