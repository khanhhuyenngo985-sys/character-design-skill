
### 群像镜头

## 色彩稳定性检查

```
□ 角色主色饱和度 ≥ 背景饱和度
□ 角色色彩和环境色彩有足够色相差
□ 有光源方向辅助主体背景分离
□ 肤色角色避开灰色白色浅色背景
□ 有高饱和局部细节作为色彩锚点
□ 雾雨夜间场景有强局部光源
```

---

## 镜头设计检查

```
每个镜头必须包含：
□ 角色在画面中的精确位置（左/中/右/前/后）
□ 角色与相机的距离（远/中/近/特）
□ 角色与道具/其他角色的空间关系
□ 视线方向（看向哪里/看向谁）
□ 光源方向（侧光/顶光/底光/背光）
□ 景别（决定哪些细节可见）
```

## Prompt优化流程

```
Step 1: 初始生成 → DNA完整描述 + 参考图 + 场景要求
Step 2: 第一轮评估 → 检查面部/体型/服装/色彩/细节是否一致
Step 3: 问题定位 → 面部出错？体型出错？服装出错？色彩出错？
Step 4: 针对性修复 → 强化描述/补充数据/明确色彩/加入锚点
Step 5: 第二轮生成 → 修复版提示词 + 同参考图
Step 6: 一致性检查 → 与前后镜头对照
Step 7: 如仍有问题，返回Step 3
Step 8: 定稿输出 → 最终提示词 + 参考图打包存储
```
