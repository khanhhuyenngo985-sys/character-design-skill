## 经典角色详细分析

### 案例1：火云邪神（功夫）

**基本信息**：
- 作品：《功夫》
- 导演：周星驰
- 原型：Shadow + Threshold Guardian

**反差设计**：
```
外表：秃头/背心/短裤/拖鞋
= 普通人、路人甲、毫无威胁

实际：宇宙最强
= 徒手接子弹
= 如来神掌
```

**姿态设计**：
- 驼背 → 不自信/底层人
- 拈花指 → 反差技能
- 站桩 → 宗师风范

**服装设计**：
```
背心 = 无防护
短裤 = 随便
拖鞋 = 穷
（无任何"高手"视觉包装）
```

**设计启示**：
```
真正的高手不需要视觉包装

越是不像高手，越是可怕
越是像普通人，反差越大
```

**AI Prompt写法**：
```
火云邪神，中年男子，秃头，
面容普通/眼神空洞/嘴角下垂，
体型偏瘦/身高170cm/驼背/重心居中，
服饰：白色背心/深色短裤/塑料拖鞋，
色彩锚点：白/深灰/黑，
姿态：微微驼背/双手自然下垂/站姿懒散，
表情：面无表情/眼神冷漠/嘴角微微下撇，
光源：自然光/无特殊光影，
背景：废弃工厂/普通房间，
写实风格，16:9
```

### 案例2：三角头（Silent Hill）

**基本信息**：
- 作品：Silent Hill
- 类型：游戏/电影
- 原型：Shadow（绝对威胁）

**剪影设计**：
```
巨大三角形头部
= 一眼可辨
= 视觉极端
= 无法遗忘
```

**色彩设计**：
```
深褐色皮肤
+ 暗红色血迹/痕迹
= 恐惧与血腥
= 原始恐惧
```

**姿态设计**：
```
高大身躯
+ 缓慢移动
+ 锯齿刀
= 压倒性的威胁感
```

**设计启示**：
```
剪影比色彩更重要

一个独特的轮廓
比任何细节都容易记忆
```

**AI Prompt写法**：
```
三角头，巨型男性，
巨大三角形头部/身高3米以上/身形魁梧，
深褐色皮肤/躯干有暗红血迹/手持巨大锯齿刀，
色彩锚点：深褐/暗红/黑，
姿态：缓慢行走/重心稳定/右手握刀，
表情：无面目/三角形头部/空洞，
光源：顶灯昏暗/周围黑暗，
背景：废弃城镇/迷雾/铁锈，
暗黑风格，16:9
```

### 案例3：Lucy（赛博朋克边缘跑者）

**基本信息**：
- 作品：Cyberpunk: Edgerunners
- 类型：动画
- 原型：杀手 + 导师 + 恋人 + 天真少女

**原型叠加**：
```
杀手：冷酷/专业/致命
导师：引导大卫成长/给予智慧
恋人：大卫的爱人
天真少女：内心向往月球/有梦想
```

**姿态弧线**：
```
初期：慵懒自信/单身姿态
中期：恋爱后身体靠近/收缩
后期：黄色夹克消失=完全异化
```

**服装叙事**：
```
黄色夹克 = 母亲的遗物
= 人性/温暖/希望
= 唯一暖色

黄色夹克的闭环：
大卫母亲穿 → 大卫继承 → 最终回到Lucy手中
= 视觉叙事完成情感闭环
```

**色彩设计**：
```
渐变头发 = 内心浪漫的寄托
= 科技+浪漫
= Lucy的独特标识
```

**设计启示**：
```
服装/道具 = 叙事工具

一个道具可以承载：
- 角色历史
- 情感记忆
- 主题符号
```

### 案例4：Wednesday Adams

**基本信息**：
- 作品：Wednesday
- 类型：美剧
- 原型：Hero + Trickster + Shadow

**色彩设计**：
```
环境：万花筒般的彩色
= 青春期/疯狂/混乱

Wednesday：纯黑白
= 与环境隔绝
= 死亡/哥特/冷静
= 视觉孤儿

室友Enid：全彩/毛茸茸/社交牛逼
= Wednesday的反面
= 填补Wednesday的空缺
```

**姿态设计**：
```
直立僵硬
= 拒绝情感
= 拒绝社交
= 控制欲强

几乎无曲线
= 坚硬/冰冷
```

**道具设计**：
```
断手 = 独特标识
= 攻击性
= 黑色幽默
= 不完美但记忆点极强
```

**设计启示**：
```
色彩隔离 = 视觉孤儿

让角色色彩完全脱离环境
是最强的视觉差异化方案
```

### 案例6：威门·唐尼（瞬息全宇宙）

**基本信息**：
- 作品：Everything Everywhere All at Once
- 原型：Trickster + Shapeshifter + Ally

**姿态设计**：
```
初期：驼背/懒散
= 失败者/废物

后期：挺直/战斗姿态
= 英雄觉醒
```

**道具设计**：
```
眼镜 = DNA锚点
= 全片不变
= 无论哪个宇宙
= 眼镜=威门
```

**设计启示**：
```
道具锚点 = 最强一致性

一个简单的道具
可以跨越所有平行宇宙
保持角色辨识度
```

## 角色设计 Checklist

### 群像设计

```
□ 是否绘制了完整的群像图？
□ 每个角色的色彩是否可远距离区分？
□ 每个角色的体型/轮廓是否有差异？
□ 配角是否有自己的性格特征？
□ 主角和配角的视觉权重是否拉开？
```

### DNA设计

```
□ 面部几何是否描述清晰？
□ 体型数据是否有量化（身高/肩宽）？
□ 服装主色/辅色/点缀色是否明确？
□ 标志性道具是否设计？
□ 姿态习惯是否描述？
```

### 原型设计

```
□ 每个角色是否有至少2个原型叠加？
□ 原型叠加是否有反差（不矛盾）？
□ 矛盾性是否让角色更真实？
□ 原型是否与角色弧线匹配？
```

### 色彩设计

```
□ 角色主色是否高饱和？
□ 角色色彩与环境是否有对比？
□ 是否有色彩锚点（局部高饱和细节）？
□ 色彩是否符合角色性格？
```

### 道具/服装设计

```
□ 服装是否有叙事功能（成长弧线）？
□ 道具是否是角色延伸人格？
□ 道具是否能成为记忆点？
□ 服装材质是否符合世界观？
```
