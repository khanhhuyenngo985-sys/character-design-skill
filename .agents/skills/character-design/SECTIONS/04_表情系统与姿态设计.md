# 表情系统与姿态设计

> 写实角色设计中的表情联动与姿态习惯。

---

## 情绪-表情-姿态联动

### 核心概念

情绪、表情、姿态是三个互相咬合的齿轮。在AI视频生成时，必须让三者同时出现在Prompt中，才能让角色的情绪传达准确。

### 八种核心情绪联动矩阵

| 情绪 | 眼睛 | 眉毛 | 嘴巴 | 姿态 |
|------|------|------|------|------|
| **愤怒** | 缩小、下压 | +30°到+60°上扬（眉尾） | 锯齿型/横线抿紧 | 身体前倾、拳头紧握、重心前移 |
| **悲伤** | 瞳孔下垂 | +20°到+40°下压（眉头） | 下垂型 | 肩膀内收、身体下沉、重心偏下 |
| **恐惧** | 瞳孔放大、高光消失 | 抬起 | O型微张 | 身体后退、双臂护胸、重心后移 |
| **惊讶** | 瞳孔放大 | -30°到-60°下沉（眉尾） | O型 | 身体僵住、双手展开、重心居中 |
| **厌恶** | 微眯 | 下压 | 上翘 | 身体后仰、头部后移、偏头 |
| **轻蔑** | 眯眼 | 不对称 | 单边上扬 | 身体后仰、单手插兜、偏头 |
| **喜悦** | 弯月形 | 上扬 | 上扬型 | 身体放松、肩膀抬起、重心居中 |
| **平静** | 正常 | 水平（-5°到+5°） | 横线型 | 身体放松、重心居中、肌肉松弛 |

### 愤怒三型（最常用）

```
压抑型：嘴角抿成直线，拳头握紧垂在身侧
爆发型：川字眉，牙齿咬紧，身体前倾双拳举起
威压型：眉心无皱纹，嘴角微微下压，身体静止不动
```

### 复合情绪表达

现实中情绪很少单一出现，更多是复合情绪：

| 复合情绪 | 面部表现 | 姿态表现 |
|---------|---------|---------|
| 愤怒+悲伤 | 眉心皱缩 + 眼眶泛红 | 身体颤抖 |
| 恐惧+愤怒 | 瞪大双眼 + 咬牙切齿 | 防御姿态 |
| 惊讶+喜悦 | 眉眼上扬 + 嘴角上扬 | 身体前倾 |
| 厌恶+轻蔑 | 眯眼 + 嘴角单边上扬 | 偏头后仰 |
| 悲伤+愤怒 | 眉头紧皱 + 下唇颤抖 | 双拳握紧 |

## 表情细节参数

### 眼睛8态参数

| 眼睛状态 | 瞳孔位置 | 眼皮状态 | 高光位置 | 用于表达 |
|----------|----------|----------|----------|----------|
| 正常 | 瞳孔居中 | 正常 | 右上 | 基准状态 |
| 惊讶 | 瞳孔放大 | 眼皮抬起 | 缩小 | 震惊、呆住 |
| 愤怒 | 瞳孔缩小 | 眼皮下压 | 缩小或消失 | 生气、敌意 |
| 恐惧 | 瞳孔放大 | 眼皮完全睁开 | 消失 | 惊恐、惊吓 |
| 得意 | 瞳孔缩小 | 眼皮下压 | 右上 | 嚣张、得意 |
| 悲伤 | 瞳孔略下垂 | 上眼皮上抽 | 缩小 | 委屈、难过 |
| 斗鸡眼 | 向中心聚集 | — | — | 懵圈、困惑 |
| 呆萌 | 瞳孔缩小 | 正常 | 正常 | 平静、呆滞 |

### 眉毛角度密码

以水平线为0°，测量眉毛与水平线的夹角：

| 情绪 | 角度范围 | 眉毛形态描述 |
|------|----------|-------------|
| 愤怒 | +30°到+60° | 眉尾上扬，眉头下压，呈"倒八"形 |
| 惊讶 | -30°到-60° | 眉尾下沉，眉头上扬，呈"正八"形 |
| 悲伤 | +20°到+40° | 眉头下压，眉尾相对稳定 |
| 担忧 | +10°到+20° | 眉头微皱，眉毛整体上扬 |
| 平静 | -5°到+5° | 几乎水平，无明显起伏 |
| 专注 | 0°到+15° | 眉头微皱，专注思考 |

### 嘴巴6种变化

| 嘴巴形状 | 适用情绪 | 视觉特征 |
|----------|----------|----------|
| O型 | 惊讶、张嘴 | 嘴唇圆形张开 |
| 横线型 | 抿紧、生气 | 嘴唇平直紧绷 |
| 上扬型 | 微笑、得意 | 嘴角向上弯曲 |
| 下垂型 | 沮丧、委屈 | 嘴角向下弯曲 |
| 锯齿型 | 愤怒、大喊 | 嘴唇边缘呈锯齿状 |
| 月牙型 | 傻笑、开心 | 嘴角大弧度上弯 |

### 面部肌肉关联

```
皱眉肌收紧 → 眉心皱纹 + 眼睛变小
颧大肌收紧 → 嘴角上扬 + 脸颊提升
眼轮匝肌收紧 → 眼睛闭合 + 眼角皱纹
降眉间肌收紧 → 眉头下压 + 鼻梁皱纹
```

## 姿态设计

### 姿态公式

```
姿态 = 重心位置 × 四肢角度 × 脊椎曲直 × 头部位置
```

### 三原则

1. **重心**：高重心=自信（直立），低重心=压抑（下沉）
2. **四肢**：张开=开放，紧缩=封闭
3. **脊椎**：直=力量，弯=谦卑/疲惫

### 姿态与原型关系

| 性格/阶段 | 姿态特征 |
|----------|---------|
| 自信状态 | 身体舒展，重心居中 |
| 恋爱/亲密状态 | 身体靠近，动作收缩 |
| 战斗/对抗状态 | 身体前倾，占据空间 |
| 疲惫/压抑状态 | 重心前倾，背部弓形 |
| 反派/压迫感 | 俯视角度，身体收缩 |

### 姿态习惯设计

每个角色应有独特的姿态习惯：

```
站姿类型：
- 挺拔直立（自信/军人）
- 微微驼背（压抑/疲惫/老人）
- 三七分重心（放松/随意）
- 丁字步（传统/礼仪）

手部习惯：
- 双手插兜
- 抱胸
- 背手
- 自然下垂
- 叉腰

坐姿类型：
- 端坐（正式/紧张）
- 瘫坐（放松/懒散）
- 翘二郎腿
- 身体前倾
```

## 视线与眼神设计

### 视线方向叙事

| 视线方向 | 叙事含义 |
|---------|---------|
| 看镜头（看观众） | 直面交流，邀请参与 |
| 看角色A | 关注角色A，观众被引导看A |
| 看画面外左侧 | 画面外有信息，引发好奇 |
| 看画面外右侧 | 回忆/期待 |
| 低头/看地 | 内省、压抑、失败、羞愧 |
| 仰头/看天 | 希望、祈祷、放弃、死亡 |
| 斗鸡眼 | 懵圈、喜剧效果 |
| 瞳孔高光消失 | 死亡、精神崩溃 |

### 眼神与情绪

```
坚定眼神：瞳孔居中，目光稳定，直视目标
游离眼神：瞳孔缓慢移动，思考/分神
聚焦眼神：瞳孔收缩，专注某物
涣散眼神：瞳孔放大，目光失焦
回避眼神：看向别处，不愿直视
对视：直接目光接触，对抗/亲密
```

## 服装与姿态的关系

### 服装影响姿态

```
紧身服装 → 姿态受限，动作克制
宽松服装 → 姿态自由，动作舒展
正装 → 身体挺直，动作正式
休闲装 → 身体放松，动作随意
```

### 姿态影响服装呈现

```
挺胸 → 胸部服装紧绷
驼背 → 背部布料堆积
前倾 → 服装前摆拉紧，后摆多余
后仰 → 服装后摆拉紧，前摆多余
```

## 配饰与姿态

### 配饰对姿态的影响

```
眼镜：推眼镜动作（思考/确认）
帽子：扶帽檐动作（确认/掩饰）
围巾：拉紧/整理（保暖/不安）
手套：摘手套动作（准备/仪式感）
背包：调整肩带动作（负担/准备）
```

### 配饰的视觉锚点作用

配饰不仅影响姿态，更是角色DNA的一部分：

```
配饰设计要点：
□ 位置精确（左耳/右手/颈部）
□ 形状具体（圆形/方形/十字架）
□ 材质明确（金属/皮革/布料）
□ 色彩独特（与服装形成对比）
□ 尺寸比例（过大/过小/适中）
```
