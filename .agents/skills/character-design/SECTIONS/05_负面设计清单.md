# 负面设计清单

> 按阶段分类：设计阶段 → 提示词阶段 → 生成阶段 → 后期阶段。
> 适用于：AI图像/视频生成、角色立绘、短剧角色策划。

---

## 一、设计阶段错误

### ❌ 错误1：只做主角，不做群像

| 项目 | 内容 |
|------|------|
| 严重等级 | 🔴 致命（会导致后续所有镜头角色撞型） |
| 现象 | 只给主角出设计，配角随便混过去。结果主角和配角撞型、撞色、撞脸。 |
| 后果 | 观众分不清谁是谁；主角辨识度降低；AI视频多角色镜头全融在一起 |
| 修复优先级 | ★★★ 最高（必须重做群像，无法挽救） |

**错误示例**：
```
只设计主角：
- 主角：帅哥，黑色长发，蓝色外套
- 配角：随便混混，反正镜头少

结果：
- 配角A和配角B撞脸
- 配角和主角撞色
- 观众看完全片分不清谁是谁
```

**正确做法**：先出群像，再从群像里提取个体。每个角色必须有不可替代的色彩/体型/姿态。

**群像设计检查**：
```
□ 是否绘制了完整的群像图？
□ 每个角色的色彩是否可远距离区分？
□ 每个角色的体型/轮廓是否有差异？
□ 配角是否有自己的性格特征？
□ 主角和配角的视觉权重是否拉开？
```

---

### ❌ 错误2：原型单一化

| 项目 | 内容 |
|------|------|
| 严重等级 | 🟠 高（角色扁平，但不影响生成成功率） |
| 现象 | 主角 = 纯英雄，配角 = 纯阴影，没有原型混搭 |
| 后果 | 角色扁平无聊，观众记不住 |
| 修复优先级 | ★★ 高（影响角色记忆度，但不直接导致生成失败） |

**错误示例**：
```
角色原型设计：
- 主角：纯Hero（勇敢、正义、成长）
- 反派：纯Shadow（邪恶、坏、压迫）
- 配角：纯Ally（帮助、鼓励）

问题：
- 角色太脸谱化
- 观众提前猜到角色发展
- 缺乏记忆点
```

**正确做法**：每个主要角色至少叠加两个原型。

**原型叠加示例**：
```
Lucy = 杀手 + 导师 + 恋人 + 天真少女（内心）

艾伦·耶格尔 = Hero + Shadow（被使命撕裂）

老乞丐 = Mentor + Shadow（反差型导师）

威门·唐尼 = Trickster + Shapeshifter（看似废柴实则强大）
```

---

### ❌ 错误3：剪影无辨识度

| 项目 | 内容 |
|------|------|
| 严重等级 | 🟡 中（影响角色辨识度，不直接影响生成） |
| 现象 | 换个颜色/发型就能变成另一个角色 |
| 测试方法 | 只给AI看剪影（纯黑轮廓），问"能认出是谁吗"？如果不能，说明剪影设计失败 |
| 修复优先级 | ★ 中（可以后续修复，不致命） |

**剪影设计原则**：
```
好剪影 = 独特轮廓几何
□ 头部形状（圆/方/尖/不规则）
□ 肩部线条（宽/窄/斜/平）
□ 服装外轮廓（紧身/宽松/层叠）
□ 道具剪影（剑/杖/枪/无）
```

**测试流程**：
1. 生成角色剪影图（纯黑轮廓）
2. 拿给不了解角色的人看
3. 问"能认出是谁吗"
4. 如果不能 → 剪影设计失败

---

### ❌ 错误4：角色太"完美"

| 项目 | 内容 |
|------|------|
| 严重等级 | 🟡 中（影响角色记忆度，不影响生成） |
| 现象 | 主角 = 帅哥/美女，善良正直，无缺陷 |
| 后果 | 角色没有记忆点，无法形成文化符号 |
| 修复优先级 | ★ 中（可以后续修复，不致命） |

**完美角色的问题**：
```
完美 = 无差异化 = 无记忆点
- 观众看完就忘
- 无法形成表情包/二创
- 缺乏话题性
```

**正确做法**：角色必须有缺陷、有争议、有矛盾。

**有缺陷的角色示例**：
```
Wednesday：反社会、冷漠、拒绝社交
- 但：可爱、聪明、有内心柔软

Lucy：杀手、冷酷、边缘人
- 但：内心有小女孩梦想、渴望温暖

电锯人：暴力、无情、混乱
- 但：天真、搞笑、有普通人的欲望
```

---

### ❌ 错误5：服装无叙事

| 项目 | 内容 |
|------|------|
| 严重等级 | 🟡 中（影响角色深度，不影响生成） |
| 现象 | 服装只考虑"好看"，不考虑叙事功能 |
| 后果 | 服装浪费了叙事机会 |
| 修复优先级 | ★ 中 |

**服装四大叙事功能**：
1. **造型**：人为制造尖锐/圆润边缘，改变剪影
2. **叙事**：承载角色成长弧线
3. **身份认同**：纹样代表门派/阵营/文化
4. **锚定**：固定道具让AI记住角色

## 二、提示词阶段错误

### ❌ 错误6：提示词里不写角色DNA

| 项目 | 内容 |
|------|------|
| 严重等级 | 🔴 致命（直接导致角色外观随机化） |
| 反面示例 | `"一个英俊的男人走在街上" ❌` |
| 原因 | 模型不认识"英俊"，也无法稳定复现一张抽象的脸 |
| 修复优先级 | ★★★ 最高（必须重写提示词，加DNA标签） |

**反面案例分析**：
```
❌ "一个英俊的男人走在街上"

问题：
- 模型不知道"英俊"长什么样
- 每一次生成都是随机脸
- 角色不稳定，无法形成记忆
```

**正确做法**：每条提示词必须包含角色DNA标签，至少3项：

```
✅ "角色A，
颧骨高下巴尖（面部几何）
短发右边分（发型）
标志性红色围巾（道具）
黑色皮外套（服装）
微微驼背（姿态）
..."

面部几何 = 脸型/颧骨/眉形/眼形/鼻形/嘴形/下巴
体型 = 身高/肩宽/体型/重心位置
服装 = 主色/辅色/标志性单品
细节 = 疤/痣/眼镜/配件
```

### ❌ 错误7：多人场景不写角色区分

| 项目 | 内容 |
|------|------|
| 严重等级 | 🔴 致命（直接导致多人镜头角色融脸） |
| 反面示例 | `"三个人在聊天" ❌` |
| 后果 | 三个人的脸会互相渗透，变成同一张脸的三种角度 |
| 修复优先级 | ★★★ 最高（必须重写提示词，加角色区分） |

**反面案例**：
```
❌ "三个人在聊天"

失败表现：
- 三个人并排站
- 脸型几乎一样
- 服装颜色相近
- 观众分不清谁是谁
```

**正确写法**：
```
✅ 角色A（高个/短发/红色外套）站在画面左侧
角色B（矮个/长发/蓝色外套）站在画面中间
角色C（中等/卷发/绿色外套）站在画面右侧

✅ 角色A面朝右，角色B面朝左，角色C面朝前方
✅ 三人之间各保持半身位距离
```

---

### ❌ 错误8：不写光影方向

| 项目 | 内容 |
|------|------|
| 严重等级 | 🟠 高（导致同一角色在不同镜头光影不一致） |
| 反面示例 | `"两个人在室内对话" ❌` |
| 后果 | 镜头1角色脸部右侧受光，镜头2同一角色脸部左侧受光，观众感觉是两个人 |
| 修复优先级 | ★★ 高（必须在每条提示词里加光影方向描述） |

**反面案例**：
```
❌ "两个人在室内对话"

失败表现：
- 镜头1：角色A脸部右侧受光
- 镜头2：角色A脸部左侧受光
- 镜头3：顶光
- 观众：这是同一个人吗？
```

**正确做法**：
```
✅ "两个人在室内对话，
右上方45°侧光（稳定光源方向），
角色面部亮侧朝镜头。"

全片统一光影方向描写。
```

**常用光影方向**：
```
右上方45°侧光：标准电影光
左上方45°侧光：标准电影光
正面柔光：温馨、平静
顶光：审讯、压迫
底光：恐怖、不正常
背光：孤独、剪影
```

---

### ❌ 错误9：服装色彩太接近

| 项目 | 内容 |
|------|------|
| 严重等级 | 🟠 高（导致同屏角色难以区分） |
| 反面示例 | `"主角穿深蓝外套，配角1深灰外套，配角2藏蓝外套" ❌` |
| 后果 | 三个深色角色同屏，观众分不清谁是主角 |
| 修复优先级 | ★★ 高（必须重新设计色彩方案，拉开色彩差距） |

**反面案例**：
```
❌ 色彩太接近：
- 主角：深蓝色外套
- 配角1：深灰色外套
- 配角2：藏蓝色外套

失败表现：
- 三个深色角色挤在一起
- 观众找不着主角
- 视觉混乱
```

**正确做法**：
```
✅ 色彩隔离：
- 主角 = 高饱和暖色（红/橙/黄）
- 配角 = 低饱和冷色（灰/蓝/绿）

✅ 或者：
- 主角 = 单一突出色
- 配角 = 混合色/无色彩
```

**AI稳定色彩方案**：
```
1. 主体高饱和 + 背景中性灰      ← 最安全
2. 主体高饱和 + 背景互补色      ← 最强冲击
3. 主体深色 + 背景高亮度        ← 明度差异
4. 主体单色 + 配饰高饱和锚点    ← 最实用
```

### ❌ 错误10：姿态永远站桩

| 项目 | 内容 |
|------|------|
| 严重等级 | 🟡 中（影响视频观感，不直接导致生成失败） |
| 反面示例 | `"角色立绘，正面站立，双手自然下垂" ❌` |
| 后果 | 视频里角色永远是立正，没有性格传达 |
| 修复优先级 | ★ 中（影响观感，可以后续补姿态参考图） |

**正确做法**：每个角色必须给3种姿态参考图：

```
1. 日常态（放松）：重心居中，肌肉松弛
2. 紧张态（战斗/压力）：重心前倾，肌肉紧绷
3. 亲密态（信任/开放）：身体微倾，开放姿态
```

## 三、生成阶段错误

### ❌ 错误11：不传参考图/三视图不齐全

| 项目 | 内容 |
|------|------|
| 严重等级 | 🔴 致命（直接导致侧脸/背影不像同一人） |
| 现象 | 只给一张正面照就想让模型保持一致 |
| 后果 | 侧脸/背影/换服装时角色完全变样 |
| 修复优先级 | ★★★ 最高（必须补齐三视图，重新生成侧脸/背影） |

**三视图标准配置**：
```
1. 正脸图（正面头像/上半身）
   - 看清面部特征
   - 看清发型、额头
   - 建立基础印象

2. 侧脸图（90°侧脸）
   - 看清颧骨、下巴轮廓
   - 看清鼻子高度
   - 侧脸是角色辨识关键

3. 全身图（站立姿态）
   - 看清体型比例
   - 看清重心位置
   - 看清服装整体
```

**参考链建立**：
```
前一镜输出图 → 下一镜参考 + 三视图
```

### ❌ 错误12：景别跳跃太大

| 项目 | 内容 |
|------|------|
| 严重等级 | 🟠 高（导致特写时角色重构） |
| 现象 | 上一镜是远景全身，下一镜直接切特写 |
| 后果 | 角色脸部特征在特写里完全重构，像另一个人 |
| 修复优先级 | ★★ 高（必须加中景作为过渡，或传前一镜输出图） |

**景别跳跃规则**：
```
✅ 远景 → 中景 → 近景 → 特写（渐次推进OK）
❌ 远景 → 特写（跳跃太大）

✅ 特写 → 近景 → 中景 → 远景（渐次拉远OK）
❌ 特写 → 远景（跳跃太大）
```

**修复方案**：
```
方案1：加中景过渡
远景全身 → 中景膝盖以上 → 近景胸口以上 → 特写脸部

方案2：传前一镜输出图
远景全身 → 传远景图+特写Prompt → 特写脸部
```

---

### ❌ 错误13：运动强度过高

| 项目 | 内容 |
|------|------|
| 严重等级 | 🟠 高（导致角色结构崩溃） |
| 现象 | 打斗/奔跑等大动作选"强运动" |
| 后果 | 角色衣服颜色消失、头发乱飞、脸部变形 |
| 修复优先级 | ★★ 高（必须降低运动强度，分段处理大动作） |

**运动强度指南**：
```
弱运动：头发轻微飘动、衣物摆动、面部表情变化
中运动：走路、挥手、转身、慢跑
强运动：快跑、跳跃、激烈手势
极强运动：打斗、飞行、高速追逐 ← 风险最高
```

**分段处理大动作**：
```
❌ 一个镜头：主角打斗全套动作
✅ 多个镜头：
   镜头1：预备动作（准备姿势）
   镜头2：出拳（弱-中运动）
   镜头3：击中（中运动）
   镜头4：反应（弱运动+表情特写）
```

---

### ❌ 错误14：同一角色多人大全镜不拆

| 项目 | 内容 |
|------|------|
| 严重等级 | 🟠 高（导致多人同屏全部变形） |
| 现象 | 5人以上同框全部给正面特写 |
| 后果 | 5个人的脸挤在一个画面里，全部变形/融脸/撞型 |
| 修复优先级 | ★★ 高（必须拆分镜头，大全镜只给远景/剪影） |

**多人场景规则**：
```
大全镜（5人以上）：只给远景/剪影/背影，不给面部特写
中景（3-4人）：确保体型/色彩差异足够大，不给特写
近景（2人）：可给面部特写
特写（1人）：单人面部特写
```

## 四、后期阶段错误

### ❌ 错误15：不检查就过

| 项目 | 内容 |
|------|------|
| 严重等级 | 🔴 致命（累积误差会导致第10镜角色面目全非） |
| 现象 | 角色脸变了、衣服颜色变了，但觉得"差不多能用"就过了 |
| 后果 | 累积误差越来越大，第10个镜头时角色已经面目全非 |
| 修复优先级 | ★★★ 最高（必须逐镜检查，失败就重生成） |

**检查频率**：
```
每个镜头输出后，必须检查：
□ 面部特征是否一致？
□ 服装色彩是否一致？
□ 发型是否一致？
□ 体型比例是否一致？
□ 光影方向是否一致？

通过 → 下一镜
失败 → 重新生成
```

---

### ❌ 错误16：角色标志性细节丢失

| 项目 | 内容 |
|------|------|
| 严重等级 | 🟠 高（影响角色辨识度） |
| 现象 | 某个镜头里角色的眉疤/耳钉/胎记/标志性配饰消失了 |
| 原因 | 提示词没写，模型在生成时把这些细节当"噪声"过滤了 |
| 修复优先级 | ★★ 高（必须在提示词里强制写标志性细节） |

**标志性细节列表**：
```
□ 眉疤/眉形特征
□ 耳钉/耳环
□ 胎记/痣
□ 眼镜/眼罩
□ 项链/项圈
□ 腰带/特殊配件
□ 发型特征（鬓角/刘海/发色）
□ 服装特殊设计（补丁/破洞/徽章）
```

**提示词格式**：
```
✅ "角色A，眉间有横向刀疤（必须保留）"
✅ "角色A，左耳有银色耳钉"
❌ "角色A"（无细节描述）

遮挡情况：
✅ "角色A，眉间刀疤被刘海遮挡"
✅ "角色A，左耳耳钉已摘下（当前状态）"
```

## 快速检查表（每镜必查）

### 设计阶段

```
□ 群像先于个体设计？
□ 每个角色至少2个原型混搭？
□ 剪影有辨识度（测试过）？
□ 角色有缺陷/矛盾点？
□ 服装有叙事功能？
□ 每个角色有不可替代的色彩/体型？
```

### 提示词阶段

```
□ DNA标签≥3项（面型/体型/服装/细节）？
□ 多人场景写了色彩区分？
□ 光影方向锁定（45°侧光/顶光/底光）？
□ 姿态不是站桩（有重心/四肢描述）？
□ 标志性细节被提及？
□ 视线方向精确描述？
```

### 生成阶段

```
□ 三视图参考图齐全？
□ 景别跨度合理（远→中→近OK）？
□ 运动强度匹配动作类型？
□ 多人场景≤3人同镜？
□ 参考图是同一角色的同一状态？
```

### 后期阶段

```
□ 标志性细节（眉疤/耳钉/胎记）都在？
□ 服装主色没变？
□ 脸型一致（用三视图对比）？
□ 光影方向一致？
□ 发型没有改变？
□ 体型比例一致？
```

## 错误→症状→修复方案

| 错误 | 症状 | 修复方案 |
|------|------|---------|
| 不写DNA | 每次生成脸不一样 | 加DNA标签 |
| 不写光影 | 同一角色光影不一 | 锁定光源方向 |
| 不传参考图 | 侧脸/背影变样 | 补三视图参考 |
| 景别跳跃 | 特写时角色重构 | 加过渡镜头 |
| 运动过强 | 角色变形崩溃 | 降低运动强度 |
| 不检查 | 累积误差爆炸 | 每镜必查 |
| 细节丢失 | 标志性特征消失 | 提示词强制写明 |
