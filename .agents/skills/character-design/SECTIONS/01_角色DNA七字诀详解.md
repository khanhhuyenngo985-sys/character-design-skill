# 角色DNA七字诀详解

> 设计角色的七个维度。全部用"差异性"贯穿。

---

## 一、群像优先

### 核心原则

**核心**：先设计群像，再提取个体。角色A的"善良"必须由角色B的"邪恶"来衬托。

### 受众推导链

```
目标受众 → 他们的审美偏好 → 他们见过什么 → 他们期待什么 →
什么元素能让他们"意外" → 反差设计方向
```

### DNA原则（防止AI融合）

```
发型：高/矮 → 短发/长发 → 颜色差异（黑/金/红）
体型：高/矮 → 胖/瘦 → 肩宽/窄
脸型：圆/方/椭圆/长
肤色：白/黄/黑/棕
服装：职业装/休闲/制服
色彩：暖/冷/对比/单色
```

### 视觉领地象限图

将角色放置在"色彩冷暖"与"视觉复杂度"组成的象限中。任何两个角色不能占据同一个象限。

| 象限 | 特征组合 | 典型案例 | 视觉传达 |
|------|---------|---------|---------|
| **I** | 复杂 + 暖调 | 大卫 (赛博朋克) | 充满细节、热血、处于冲突中心 |
| **II** | 简单 + 暖调 | 曼恩 / 露西 (后期) | 沉稳、母性/父性、力量感、可信任 |
| **III** | 简单 + 冷调 | 露西 (初期) / 星期三 | 疏离、神秘、局外人、干练 |
| **IV** | 复杂 + 冷调 | 亚当·重锤 / 最终反派 | 机械压迫、非人化、精密而致命 |

### AI多人相融问题预防（在设计阶段）

1. **色块差异**：每个主要角色分配专属色彩标识，确保视觉远看可区分
2. **体型差异**：高矮胖瘦、身型轮廓要有明显差别
3. **服装差异**：材质感、复杂程度、风格标签各不相同
4. **姿态差异**：驼背 vs 挺拔，展开 vs 收缩

## 二、原型混搭

### 八种原型

| 原型 | 核心功能 | AI设计优先级 |
|------|---------|--------------|
| **Hero（英雄）** | 推动叙事弧线 | 成长弧线 |
| **Shadow（阴影）** | 制造最大冲突 | 威胁感 |
| **Mentor（导师）** | 给予智慧/道具 | 标志性细节 |
| **Trickster（诡计者）** | 打破规则，制造意外 | 姿态反差 |
| **Threshold Guardian（阈限守卫）** | 测试Hero | 弱点设计 |
| **Herald（信使）** | 宣告/召唤 | 视觉信号 |
| **Shapeshifter（变形者）** | 制造悬念 | DNA锚点 |
| **Ally（伙伴）** | 提供支持 | 独立性 |

### 原型配置公式（按受众目标定制）

| 目标受众 | 强化哪些原型 |
|---------|------------|
| 儿童 | 吉祥物（宠物/吉祥物型导师）+ 导师 |
| 成年男性 | 硬汉（四肢发达不服就干）+ 战略家（深思熟虑，琅琊榜型）|
| 青少年 | 劲敌 + 恋爱/暗恋对象 |
| 成年女性 | 闺蜜 + 直男对照组 |
| 家庭观众 | 长辈 + 宠物 |

### 原型叠加示例

两个原型叠加在同一角色上，差异性翻倍。

```
Hero + Shadow = 被使命撕裂的悲剧英雄（艾伦·耶格尔）
Hero + Mentor = 用死亡教育后辈的牺牲者（欧比旺）
Shadow + Ally = 表面敌人实际帮助主角（三角头）
Mentor + Shadow = 反差型导师，平日是普通老头（老乞丐/火云邪神）
```

### 经典角色原型分析

**Lucy（赛博朋克边缘跑者）**
- 原型叠加：杀手 + 爱人 + 导师 + 神秘人
- 初期：慵懒自信的单身姿态
- 恋爱后：强烈母性光辉（实为大卫死去母亲的镜像角色）
- 黄色夹克的服装闭环：视觉叙事完成了角色原型的传承

**David（赛博朋克边缘跑者）**
- 原型叠加：朋克小子 + 恋人 + 母亲形象的保护对象 + 战斗者
- 姿态弧线：驼背（自卑）→ 挺直（自信）→ 僵硬（完全异化）
- 色彩：黄色夹克是人性/母亲/感情的象征

## 三、姿态叙事

### 姿态公式

```
姿态 = 重心位置 × 四肢角度 × 脊椎曲直 × 头部位置
```

### 三原则

1. **重心**：高重心=自信（直立），低重心=压抑（下沉）
2. **四肢**：张开=开放，紧缩=封闭
3. **脊椎**：直=力量，弯=谦卑/疲惫

### 姿态与原型关系

| 性格/阶段 | 姿态特征 |
|----------|---------|
| 自信单身状态 | 慵懒开放，重心随意 |
| 恋爱/亲密状态 | 身体靠近，动作收缩 |
| 战斗/对抗状态 | 挺拔张扬，占据空间 |
| 老人/疲惫状态 | 重心前倾，背部弓形 |
| 反派/压迫感 | 俯视、收缩、黑暗侵占空间 |

### 成长弧线姿态变化

```
阶段1：驼背（重心不稳）→ 阶段2：直立（找到方向）→ 阶段3：略微僵硬（机甲覆盖人性）
```

### 群像姿态设计规则

- 驼背 / 挺拔 / 展开 → 各有分配
- 圆形体型 / 细长体型 / 尖锐体型 → 各有分配
- 姿态要**反映角色复杂性**，而非一味追求美型

## 四、情绪-表情-姿态联动（必修）

### 核心概念

情绪、表情、姿态是三个互相咬合的齿轮。AI视频里必须让三者同时出现在Prompt中。

### 八种核心情绪联动矩阵

| 情绪 | 面部变化 | 姿态变化 |
|------|---------|---------|
| **愤怒** | 眉心皱缩/嘴角下压/颏肌紧绷 | 身体前倾/拳头紧握/重心前移 |
| **悲伤** | 眉尾上翘/嘴角下弯/眼睑下垂 | 肩膀内收/身体下沉/重心偏下 |
| **恐惧** | 眼窝张大/眉毛抬起/嘴唇微张 | 身体后退/双臂护胸/重心后移 |
| **惊讶** | 眉毛抬起/眼睛张大/O型嘴 | 身体僵住/双手展开/重心居中 |
| **厌恶** | 眉毛下压/眼睛微眯/嘴角上翘 | 身体后仰/头部后移/偏头 |
| **轻蔑** | 眉毛不对称/眯眼/嘴角单边上扬 | 身体后仰/单手插兜/偏头 |
| **喜悦** | 眉毛上扬/眼睛弯月形/嘴角上扬 | 身体放松/肩膀抬起/重心居中 |
| **平静** | 眉毛水平/眼睛正常/嘴角横线 | 身体放松/重心居中/肌肉松弛 |

### 愤怒三型（最常用）

```
压抑型：嘴角抿成直线，拳头握紧垂在身侧
爆发型：川字眉，牙齿咬紧，身体前倾双拳举起
威压型：眉心无皱纹，嘴角微微下压，身体静止不动
```

### 眼睛8态参数

| 眼睛状态 | 瞳孔 | 眼皮 | 高光 | 用于表达 |
|----------|------|------|------|----------|
| 正常 | 中央 | 正常 | 右上 | 基准 |
| 惊讶 | 放大 | 睁大 | 缩小 | 震惊 |
| 愤怒 | 缩小 | 下压 | 缩小 | 生气 |
| 恐惧 | 放大 | 睁大 | 消失 | 惊恐 |
| 得意 | 眯缝 | 下压 | 右上 | 嚣张 |
| 悲伤 | 下方 | 上抽 | 缩小 | 委屈 |
| 斗鸡眼 | 向中心聚集 | — | — | 懵圈 |
| 呆萌 | 缩小 | 正常 | 正常 | peaceful |

### 眉毛角度密码（以水平为0°）

```
愤怒：+30°到+60°（眉尾上扬）
惊讶：-30°到-60°（眉尾下沉）
悲伤：+20°到+40°（眉头下压）
担忧：+10°到+20°（眉头微皱）
peaceful：-5°到+5°（几乎水平）
```

### 嘴巴6种变化

| 形状 | 用于 |
|------|------|
| O型 | 惊讶、张嘴 |
| 横线型 | 抿紧、生气 |
| 上扬型 | 微笑、得意 |
| 下垂型 | 沮丧、委屈 |
| 锯齿型 | 愤怒、大喊 |
| 月牙型 | 傻笑、开心 |

## 五、色彩暴力

### 三种类型

| 类型 | 手法 | 案例 |
|------|------|------|
| 极端饱和 | 超高饱和度 | 瑞贝卡：绿红配 |
| 极端对比 | 黑白 vs 彩虹 | 星期三：黑白 vs 室友五彩 |
| 色彩隔离 | 环境单色+角色点缀 | 夜之城：全深蓝+大卫黄色夹克 |

### 色彩心理学

| 色调 | 情感 | 使用场景 |
|------|------|---------|
| 暖色（红/橙/黄） | 热情、人性、温暖 | 主角、恋爱情节 |
| 冷色（蓝/紫） | 神秘、理性、距离 | 杀手、科学家 |
| 高饱和 | 强烈、刺激、年轻 | 反英雄、边缘角色 |
| 低饱和/灰调 | 复杂、成熟、压抑 | 成人向、复杂角色 |
| 黑色 | 克制、死亡、权威 | 反英雄、丧服 |
| 白色 | 纯洁、脆弱，空虚 | 天真、死亡前期 |

### AI视角：色彩稳定性研究

#### 为什么AI对高饱和色更稳定

```
1. 边缘检测友好
   高饱和色 → 像素值偏离均值大 → 边缘算子响应强
   → 主体轮廓在特征图上更突出

2. 注意力机制偏向
   Transformer的Self-Attention优先响应高方差区域
   高饱和像素方差大 → 自然获得更多注意力权重

3. 色彩一致性损失
   训练数据中，高饱和区域被优先保护
   低饱和区域（灰褐/雾/阴影）更容易被模糊处理
```

#### 色彩稳定性光谱表

| 色彩类型 | 稳定性 | 备注 |
|---------|--------|------|
| 纯红/纯黄/霓虹蓝 | ★★★★★ | AI最优先，红黄是视觉极点 |
| 深红/橙色/电紫 | ★★★★ | 饱和但有深色压底 |
| 正蓝/绿色 | ★★★ | 蓝饱和但偏冷，绿在自然中太多 |
| 深灰 | ★★ | 低饱和，需强光影辅助 |
| 肤色/浅灰 | ★★ | 中间调，最易模糊 |
| 白色/雾效果 | ★ | 无饱和度，完全依赖光影 |

#### 高危场景（避开或加强光影）

```
1. 沙漠/沙尘 → 肤色融进沙地 → 解决：红/蓝围巾+强光轮廓
2. 雪景/雾景 → 浅色服装消失 → 解决：深色或饱和色
3. 夜店昏暗 → 面部模糊 → 解决：霓虹光打脸/反光材质
4. 森林草木 → 绿棕角色像长在树上 → 解决：红橙色服装
5. 赛博雨夜 → 边缘崩塌 → 解决：霓虹服装+局部打光
```

#### 五种AI稳定色彩结构

```
1. 主体高饱和 + 背景中性灰      ← 最安全基础结构
2. 主体高饱和 + 背景互补色      ← 最强视觉冲击
3. 主体深色 + 背景高亮度        ← 明度差异而非饱和度
4. 主体低饱和 + 背景高饱和      ← 反向（黑白主角vs高饱和幻觉）
5. 主体单色 + 配饰高饱和锚点    ← 最实用：灰褐角色+黄色围巾
```

#### 最大误区

⚠️ 色彩暴力 ≠ 全画面高饱和
→ 正确：结构化对比 = 主体饱和 + 环境低饱和（或互补）
→ 全身高饱和 + 背景也高饱和 = AI不知道该突出谁

## 六、服装道具

### 四大功能

1. **造型**：人为制造尖锐/圆润边缘，改变剪影
2. **叙事**：承载角色成长弧线
3. **身份认同**：纹样代表门派/阵营/文化
4. **锚定**：固定道具让AI记住角色

### 服装即角色符号

| 角色 | 色彩 | 符号意义 |
|------|------|---------|
| 麦克斯 | 红围巾 | 唯一的暖色 = 人性最后的痕迹 |
| 星期三 | 黑白 | 死亡/哥特/与周围隔绝 |
| 大卫 | 黄色夹克 | 母亲遗物 = 人性/温暖 |
| Lucy | 黄色夹克 | 人性的锚点 |

### 服装设计金律

- **记忆钩子**：给角色一个独特的、可拆卸的视觉标志（如：一只断手、一件亮黄色夹克、一个标志性项圈）
- **材质叙事**：丝绸代表阶级，破损代表经历，金属代表非人化
- **环境脱节**：让角色的装束与当前环境产生冲突（如：在舞会穿校服，在荒原穿西装），制造视觉冲击

### 道具 = 角色的延伸人格

- 选择与角色性格**反差**或**放大**的道具
- 道具可以是"宠物/伴侣型"（如星期三的断手），构成角色支持者原型的延伸

### 设计检查清单

- [ ] 服装是否反映了角色在故事不同阶段的状态变化？
- [ ] 服装材质/质感是否有区分（光滑 vs 粗糙 vs 金属感 vs 简洁 vs 繁复）？
- [ ] 道具是否能成为角色记忆点的一部分？
- [ ] 整个群像中，服装风格是否足够差异化？

## 七、极端对比

### 七种极端对比

```
1. 外表 vs 真实（普通人外表/绝世高手内心）
2. 色彩 vs 内容（可爱画风/黑暗故事）
3. 姿态 vs 身份（杀手用园艺姿态，佛像用攻击姿态）
4. 场景 vs 角色（童话世界里的黑暗角色）
5. 色彩 vs 情绪（用暖色表达悲伤/冷色表达愤怒）
6. 服装 vs 角色（正装角色在垃圾场，流浪汉在宫殿）
7. 比例 vs 真实（超大头/超小身子，制造不安感）
```

### 矛盾性原则

> 对立统一的点要出现在同一场景，让角色令人难忘

**示例**：
- Lucy：优雅 + 致命（同一场景）
- Wednesday：冷漠 + 参加舞会（反差）
- герой：专业 + 性感（多重身份）

**矛盾性检查**：
```
- 角色的A面和B面是什么？
- 这两面是否在同一场景中出现？
- 矛盾是否让角色更真实/更难忘记？
```

- **原型叠加：** 杀手 + 爱人 + 导师 + 神秘人
- **姿态：** 冷静专业，但内心有小女孩的梦想（月球）
- **色彩：** 渐变头发 = 内心浪漫的寄托
- **道具/服装：** 黄色夹克 = 人性的锚点

### David
- **原型叠加：** 朋克小子 + 恋人 + 母亲形象的保护对象 + 战斗者
- **姿态弧线：** 驼背（自卑）→ 挺直（自信）→ 僵硬（完全异化）
- **色彩：** 黄色夹克是人性/母亲/感情的象征

## 核心金句

> "我们做角色的目的不是让观众喜欢，而是让观众难忘。"

> "让观众可以进行多重解读的角色才是好角色。"

> "参考图越准确，后续提示词越少，角色一致性越高。"
