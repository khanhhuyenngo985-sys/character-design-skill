# 角色一致性维护

> AI视频逐镜头生成，无记忆。只认"参考图+提示词+空间约束"。

---

## 问题根源

```
镜头1：角色A侧脸 ✅
镜头2：角色A全身 → 模型不认识角色A → 变成另一个人 ❌
```

真正强约束 = **参考图（像素级）+ 提示词（语言级）+ 镜头设计（空间约束）**

---

## 三层锚定系统

### 第一层：参考图锚定（最强约束）

```
标准配置：角色立绘三件套
1. 正脸图（正面头像/上半身）
2. 侧脸图（90°侧脸）
3. 全身图（站立姿态）
```

**AI视频生成注意事项**：
- AI模型会根据你给的参考图"推演"角色的走路方式、声音、演技
- 姿态设定影响AI对角色动作规律的理解
- 给错误或模糊的姿态 → AI输出随机不可控的行为

### 第二层：DNA标签卡锚定（核心）

```
【角色名】
- 面部几何：[脸型/颧骨/眉形/眼形/鼻形/嘴形/下巴]
- 体型：[身高/肩宽/体型/重心位置]
- 姿态习惯：[标志性姿态/重心/站姿/坐姿]
- 服装：[主色/辅色/标志性单品]
- 色彩锚点：[主色/辅色/点缀色]
- 标志性细节：[疤/痣/眼镜/配件]
```

### 第三层：镜头设计锚定（空间约束）

```
每个镜头必须包含：
□ 角色在画面中的精确位置（左1/3/右1/3/中）
□ 角色与相机的距离（近/中/远/特写）
□ 角色与道具/其他角色的空间关系
□ 视线方向（看哪里）
```

---

## 一致性检查清单（每镜必查）

```
□ 参考图是否为同角色的同一服装/姿态？
□ 提示词中DNA标签是否完整（面型/体型/服装/细节）？
□ 镜头设计中角色位置是否与前后镜头一致？
□ 色彩锚点（主色/辅色）是否出现在提示词中？
□ 标志性细节（疤/痣/眼镜）是否被提及？
□ 光影方向是否影响角色色彩呈现？
□ 镜头距离是否影响角色细节可见度？
```

---

## 提示词优化迭代流程

```
Step 1: 初始生成
   → DNA完整描述 + 参考图 + 场景要求

Step 2: 第一轮评估
   → 检查面部/体型/服装/色彩/细节是否一致

Step 3: 问题定位
   → 面部出错？体型出错？服装出错？色彩出错？

Step 4:针对性修复
   → 面部：强化面部几何描述
   → 体型：补充身高肩宽数据
   → 服装：明确主色+辅色+材质
   → 色彩：加入色彩锚点描述

Step 5: 第二轮生成
   → 修复版提示词 + 同参考图

Step 6: 一致性检查
   → 与前后镜头对照

Step 7: 如仍有问题，返回Step 3

Step 8: 定稿输出
   → 最终提示词 + 参考图打包存储
```

## AI多人相融问题解决方案

当多个角色同框时极易相融（头发颜色相近、五官风格一致等）。

### 预防措施（在设计阶段）

1. **色块差异**：每个主要角色分配专属色彩标识，确保视觉远看可区分
2. **体型差异**：高矮胖瘦、身型轮廓要有明显差别
3. **服装差异**：材质感、复杂程度、风格标签各不相同
4. **姿态差异**：驼背 vs 挺拔，展开 vs 收缩

### 设定稿的作用

- 设定稿 = 写给AI的角色说明书
- 一张好的设定稿能让AI记住：体型、颜色、姿态、服装纹理、情绪质感
- 省去大量反复描述的提示词工作

## 角色设定标准模板

```
角色名称：【中文名】（【英文名/拼音】）

外貌特征：
- 面部特征：眼神、眉毛、表情、面部饰物等
- 发型与头饰：发色、发型、发饰、束发方式等
- 服饰细节：颜色搭配、材质质感、特殊设计、装饰图案等
- 身体特征：体型、姿态、肌肉线条等
- 特殊元素：伤痕、纹身、光环、能量流动等

职业/攻击特性：
角色的职业定位和战斗/行为风格

核心技能：
【技能一】：效果 + 视觉表现
【技能二】：效果 + 视觉表现
【技能三】：效果 + 视觉表现

武器/道具类型：
武器名称及外观、材质、象征意义

性格/风格：
- 基本性格特征
- 价值观与信念
- 行为风格
- 特殊状态（战斗/情感爆发）下的变化

视觉基准锁定（文生图核心词）：
- 整体画风：写实/卡通/水墨/赛博朋克等
- 色彩基调：主色、辅助色、强调色
- 光影效果：霓虹光/自然光/气氛光
- 特殊效果：粒子/运动模糊/流光
- 材质表现：金属光泽/布料质感/皮肤纹理
```

### 角色命名原则

| 原则 | 说明 | 示例 |
|------|------|------|
| 视觉呼应 | 与角色视觉特征相呼应 | 「赤鲤」→ 红色鳞片 |
| 技能关联 | 与技能属性相关联 | 「霜狼」→ 冰系技能 |
| 职业体现 | 体现职业特点 | 「剑魔」→ 剑客 |
| 文化底蕴 | 有一定的文化底蕴 | 「鲲鹏」→ 神话意象 |

## 跨平台一致性维护策略

### 即梦（Dreamina）一致性维护

**参考图策略**：
```
标准配置：1-3张参考图
推荐组合：正脸图 + 侧脸图 + 全身图各一张

传图顺序建议：
1. 先传正脸图（锁定面部特征）
2. 再传侧脸图（锁定轮廓）
3. 最后传姿态图（锁定身体语言）
```

**DNA描述优先级**：
```
必须写（缺一不可）：
□ 脸型（方/圆/椭圆/瓜子）
□ 颧骨（高/低/明显/平坦）
□ 眉形（粗/细/弯/直）
□ 眼形（大/小/凤眼/圆眼）
□ 鼻形（高/低/鹰钩/蒜头）
□ 嘴形（大/小/薄/厚）
□ 下巴（尖/方/圆）
```

**景别切换策略**：
```
远景 → 中景：必须写"保持角色[角色名]的面部特征不变"
中景 → 近景：必须写"保持角色[角色名]的完整面容"
近景 → 特写：必须写"锁定角色[角色名]的[具体细节，如眉疤]"
```

**色彩维护**：
```
每次生成必须包含：
- 主色（具体色相+饱和度，如"正红"、"电光蓝"）
- 辅色（具体色相，如"深灰"、"黑色"）
- 点缀色（高饱和局部细节，如"黄色头盔"、"红色围巾"
```

### 海螺（Haibo）一致性维护

**参考图策略**：
```
标准配置：1张高清正脸图（最重要）
可选：侧脸图（用于侧面镜头）

注意事项：
- 正脸图必须是高清（至少1024x1024）
- 面部光线清晰，无遮挡
- 表情为基准表情（平静或中性）
```

**DNA描述优先级**：
```
必须写（比即梦更详细）：
□ 脸型 + 颧骨 + 下颌线
□ 眉毛（具体形状+密度）
□ 眼睛（具体形状+瞳孔颜色）
□ 鼻子（具体轮廓）
□ 嘴唇（厚度+形状）
□ 皮肤质感（光滑/粗糙/油性/干性）
```

**景别切换策略**：
```
海螺对"保持"关键词响应较好
使用格式："保持角色[角色名]的面部特征一致性"
或"锁定角色[角色名]的DNA"
```

**色彩维护**：
```
海螺对高饱和色稳定性好
主推：正红、电蓝、霓虹紫、纯黄
避免：肤色与背景融化的场景
```

### 可灵（Kling）一致性维护

**参考图策略**：
```
标准配置：1-2张参考图
推荐组合：正面图 + 全身图

注意事项：
- 全身图能帮助AI理解身体比例
- 姿态图对于动作镜头尤为重要
```

**DNA描述优先级**：
```
必须写：
□ 体型（高/矮/胖/瘦/肌肉）
□ 身高（具体数值，如175cm）
□ 肩宽（宽/中/窄）
□ 重心位置（高/中/低）
□ 姿态习惯（驼背/挺直/放松）
```

**景别切换策略**：
```
可灵在远景到近景切换时一致性较差
建议：
- 远景使用"保持角色轮廓和色彩"
- 中景使用"保持角色服装色彩"
- 近景使用"保持角色面部特征"
```

**色彩维护**：
```
可灵对暖色调稳定性较好
推荐：红色系、橙色系、黄色系
注意：蓝色系在夜间场景容易偏灰

动态镜头色彩维护：
- 每5秒重新声明角色主色
- 使用"[角色名]的[主色]夹克"格式
```

## 镜头间一致性维护

### Shot-to-Shot参考链

**核心原则**：前一镜输出图 → 下一镜参考 + DNA描述

**参考链建立流程**：
```
镜头1生成完成 → 选择最佳输出作为镜头2的参考图
镜头2生成完成 → 选择最佳输出作为镜头3的参考图
以此类推...

好处：
- 保持连续性
- 减少累积误差
- AI更容易"记住"角色
```

### 位置一致性维护

**横向位置**：
```
角色位置（左/中/右）必须在每个镜头描述中明确

正确示例：
"角色A站在画面左侧1/3处"
"角色B站在画面右侧1/3处"
"角色C位于画面中央"

注意：越轴会导致角色位置混乱
```

**纵向位置**：
```
前景/中景/后景关系必须一致

正确示例：
"角色A在前景（近），角色B在后景（远）"
"两人都在中景"
```

### 视线一致性维护

**视线方向**：
```
如果角色在镜头1看向左侧，
那么在后续镜头中，除非角色转身，
否则视线方向应保持一致。

视线突然改变需要：
1. 明确的转身动作描述
2. 或时间跳转说明
```

**视点高度**：
```
角色视高（平视/俯视/仰视）应与镜头场景匹配

平视：角色眼睛与镜头齐平
俯视：镜头高于角色眼睛
仰视：镜头低于角色眼睛
```

## 常见一致性失败案例与修复

### 案例1：面部特征飘移

**问题描述**：
```
第一镜：方脸/颧骨高耸/粗眉
第三镜：变成了圆脸/低颧骨/细眉
```

**原因分析**：
- 参考图不清晰或不是同一角色
- DNA描述不完整
- 景别跳跃太大

**修复方案**：
```
1. 更换为清晰的正脸参考图
2. 补充完整DNA描述：
   "方脸/颧骨高耸/眉毛粗黑/眼睛小而狠/鼻梁挺直/嘴唇紧抿/下巴方正"
3. 景别过渡要缓和（避免远景直接跳到特写）
```

### 案例2：服装色彩改变

**问题描述**：
```
第一镜：深蓝色骑手服/黄色头盔
第三镜：变成了浅蓝色骑手服/白色头盔
```

**原因分析**：
- 没有在提示词中声明色彩锚点
- 参考图中色彩不准确

**修复方案**：
```
1. 在提示词中明确声明：
   "深蓝色骑手工作服（主色）/黄色头盔（点缀色）/黑色雨靴"
2. 每次生成都要包含这句话
3. 如色彩仍不稳定，使用"[角色名]的[颜色]服装保持不变"
```

### 案例3：体型比例失调

**问题描述**：
```
第一镜：身高175cm/肩宽/偏瘦但结实
第三镜：变成了矮胖身材
```

**原因分析**：
- 没有具体数值描述
- 参考图与文字描述不一致

**修复方案**：
```
1. 加入具体数值：
   "身高175cm/肩宽/偏瘦但结实/重心稳定"
2. 使用体型对比描述：
   "比旁边角色高半个头"
3. 全身图参考图能有效防止体型飘移
```

### 案例4：道具消失或变形

**问题描述**：
```
第一镜：角色戴着银色耳钉
第三镜：耳钉不见了
```

**原因分析**：
- 小物件在远景中容易被忽略
- 没有在DNA中声明

**修复方案**：
```
1. 在DNA标签卡中明确声明：
   "标志性细节：右耳银色耳钉"
2. 每次提示词都要提及：
   "保持角色[角色名]的银色耳钉"
3. 近景/特写镜头更容易保留小物件
```

## 季节性/光线变化下的一致性

### 日间到夜间转换

```
问题：同一角色在日间和夜间镜头中色彩差异大

解决方案：
1. 使用固定光源色温描述：
   "光源：暖黄色路灯（色温3200K）"
2. 声明角色主色不受光线影响：
   "保持角色[角色名]的[主色]服装"
3. 使用色彩补偿描述：
   "角色服装在暖光下呈现[具体色彩变化]"
```

### 室内到室外转换

```
问题：角色从室内走到室外时服装色彩变化

解决方案：
1. 室外镜头强调色彩锚点：
   "阳光照射下，角色[颜色]夹克格外显眼"
2. 使用环境对比描述：
   "在[环境色]背景下，角色[对比色]服装突出"
```
