# 白梦客影调库

将顶级摄影师/胶片/机型的"感觉"翻译成AI可复用的精确参数。

---

## 一、影调库结构

```
【影调名称】：原典出处 + 核心特征
【机型】：实际使用机型或模拟机型
【胶卷/传感】：胶片型号或数字传感特性
【色彩科学】：色彩倾向的核心逻辑
【色调配方】：高光/阴影/中间调的色彩分布
【情感特征】：这个影调传达的情绪
【AI提示词参数】：可复用的精确参数组合
```

## 二、Kodak Vision3 500T（5219）

**原典**：《1917》《银翼杀手2049》Roger Deakins
**核心特征**：冷阴影暖高光、青橙对比、电影级宽容度

### 色彩科学
- **高光偏暖**：钨丝灯色温（3200K）在高光处溢出成琥珀/橙色
- **阴影偏冷**：阴影处呈现青绿色溢出
- **对比结构**：亮部到暗部的渐进式过渡，而非截断

### 色调配方

| 区域 | 色彩 | 色值参考 |
|------|------|----------|
| 高光 | 琥珀橙 #E8A045 | 暖色溢出、发光感 |
| 中间调 | 中性灰 #A0A0A0 | 保持色彩中性 |
| 阴影 | 青绿 #2A4A4A | 冷调收缩、深度 |
| 肤色 | 暖琥珀 #D4956A | 自然暖调、不偏品 |

### AI提示词参数

```
【机型】：ARRI Alexa Mini LF（数字化Kodak Vision3）
【摄影模式】：Kodak Vision3 500T 5219 CineStill
【色温】：钨丝灯3200K + 5600K混合日光
【色调】：
  - 高光：+15 暖色溢出（amber glow）
  - 阴影：-10 品红 +15 青绿（teal push）
  - 中间调：自然中性
【对比】：柔和（filmic soft contrast）
【饱和度】：85%（轻微去饱和）
【曲线】：高光渐进压缩（highlight roll-off）
【颗粒】：轻度Kodak电影颗粒（5219风格）
【情感】：忧郁诗意、电影感、历史厚重
```
