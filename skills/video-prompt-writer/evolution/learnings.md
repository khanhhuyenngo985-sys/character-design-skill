# Video Prompt Writer 学习总结

从案例库中提取的通用模式、新技巧、优化方向。

---

## 沉淀出的模式

### D类（奇观型）常用技巧

```
- 前3秒先给城市尺度与奇观终点
- 镜头运动：俯冲/环绕/拉镜要有明确动机
- 运镜动机必须写在视听法则里
- 稳帧点是落幅那一刻
```

### E类（写实闭环型）常用技巧

```
- 动作闭环：接触→命中→结果必须完整
- 结果必须影响存在状态，不能只是见光见血
- 镜头不奉承动作，让结果自己说话
```

### F类（文戏型）常用技巧

```
- 所有情绪不能说破，只能被看见
- 微表情要具体：眼尾/嘴角/呼吸/停顿
- 暧昧藏在控制、镜像和实物交接里
```

### A类（动作群像型）常用技巧

```
- 主焦点只有一个，其他只做陪衬层
- 群像秩序要一眼看懂
- 动作配合要有因果，不能同频表演
```

## 过门方式使用统计

| 过门方式 | 使用次数 | 最佳场景 |
|----------|----------|----------|
| 动作动势硬切 | - | 追车、飞跃、撞击 |
| 拟音先入+动势延续 | - | 速度感、上一个动作的结果延续 |
| 情绪峰值硬切 | - | 台词关键点、情绪爆发 |
| 空间吞没硬切 | - | 进入新场景、裂缝吞没 |
| 动势匹配硬切 | - | 追车、连续动作 |
| 结果位移硬切 | - | 撞击、爆炸、命中 |
| 图形匹配硬切 | - | 形状/颜色呼应跳切 |

## 新发现技巧

### 来自 Seedance 2.0 CSV 的模式（2026-04-02）

#### 1. FORMAT 头标准化写法
```
FORMAT: [时长] / [镜头数] / [类型] / [对话] / [画幅]
```
变体：
- `FORMAT: 15s / ONE CONTINUOUS SHOT`（连续单镜）
- `FORMAT: 15s / 11 CUTS / HYPER-TENSION`（多切+情绪标注）
- `FORMAT: 15s / 6 SHOTS / 6 BEATS / STYLIZED 3D CEREMONIAL PAYOFF`

#### 2. 镜头时间编码规范
```
[0-3s] 或 0:00–0:02
Shot 01 (0:00–0:02)
镜头1 (0s-3s)
```
推荐用 `0:00–0:02` 格式，更清晰。

#### 3. STYLE 前置描述词
```
 Cinema look: ARRI电影机 / 8K ultra HD / 35mm film grain / 浅景深
 Style融合: 国风炫彩水墨特效 / 电影级渲染 / 高饱和金赤色流光
```
专业提示词通常在开头用一行浓缩 STYLE。

#### 4. 镜头运动术语库
| 术语 | 含义 |
|------|------|
| Steadicam | 稳定器跟拍 |
| handheld camera / 轻微手持感 | 真实感、偷拍视角 |
| tracking shot | 跟踪拍摄 |
| dolly zoom | 推拉变焦（Vertigo效果） |
| whip pan | 极速横摇 |
| orbit slow-mo | 环绕慢动作 |
| rack focus | 焦点切换 |
| POV | 主观视角 |
| crane shot | 升降拍摄 |
| push in / pull out | 推进/拉远 |

#### 5. 声音设计分层
```
Audio: 极低频宇宙轰鸣声渐起 / 沉重悠远水墨散开气浪音
音效：沉重金属撞击声 / 压抑感重低音
配乐：单鼓重击起 / 急促弦乐 / 史诗感战鼓配乐
```
专业提示词包含：环境音+动效+音乐的时间轴设计。

#### 6. 禁止项标准化
```
禁止：水印，字幕，整齐方阵，重复模型
Avoid: 夸张咆哮，打闹撕扯，低俗闹剧，卡通表情
```
禁止项通常在末尾单独成段。

#### 7. Must-Show 锚点法
每个镜头必须出现的核心元素：
```
Must-Show：神明的背影 + 单手张向虚空
Must-Show：雨滴接触脸颊的极慢动作
```
类似我们的"主语锚点"概念。

#### 8. 过门方式（Transition Types）
| 过门 | 适用场景 |
|------|----------|
| 硬切 | 追车、撞击、情绪爆发点 |
| 子弹时间切入 | 关键动作放大、细节展示 |
| 拟音先入 | 速度感延续、上一个动作结果 |
| 情绪峰值硬切 | 台词关键点 |
| 空间吞没硬切 | 进入新场景 |
| 动势匹配硬切 | 连续追车、动作衔接 |

#### 9. 镜头类型分类（已整合到 v1.1）
```
D类（奇观型）：大量视觉特效、航拍、爆炸、CG渲染
E类（写实闭环型）：真实物理反应、命中结果、动作闭环
F类（文戏型）：微表情、情绪细节、克制表演
A类（动作群像型）：主焦点唯一、群像秩序
```

#### 10. 运镜动机明确化
```
运镜动机：展示神性能量的微观凝结过程
运镜动机：揭示时间流逝的客观结果，强化孤独与超越感
```
每种镜头运动必须有叙事理由。

#### 11. 双版本锚（9:16 vs 16:9）
竖屏创作需标注构图差异：
```
9:16：人物居中，背景简化
16:9：可包含更多环境元素
```

#### 12. 质感参数库
| 类型 | 参数 |
|------|------|
| 电影级 | ARRI电影机 / 8K / 35mm film grain / anamorphic lens look |
| 超写实 | 真实皮肤质感 / 真实布料反光 / 真实物理引擎 |
| 动漫风 | 硬边水墨线条 / 青银赤金高对比 / Anime Lens Flare |
| 复古胶片 | Kodak Ektachrome 1970s / 暖黄复古色调 / 轻微胶片颗粒 |
