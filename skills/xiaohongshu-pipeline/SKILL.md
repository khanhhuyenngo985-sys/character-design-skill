> 版本：v1.0 | 更新：2026-04-07 | 适用：MJ v7 + niji7 + 即梦 Seedance 2.0

- MJ 研究**画家/美学运动**，专注静态画面美学
- Seedance 专注**动态视觉序列**，不接受情绪标签

## STEP 0 · 热点/方向确定

### 每日自动化（定时任务）

```

/last30days 小红书 AI短片 热门话题
```

### 内容方向占比

| 方向 | 占比 | 目的 |
|------|------|------|
| 情绪短片 | 50% | 建立风格辨识度 |

### 方向初选问题

```

```

## STEP 1 · 脚本策划

### 分镜数量参考

| 时长 | 分镜数 | 每段时长 |
|------|--------|---------|
| 10秒 | 3-4个 | 2-3秒/段 |
| 15秒 | 4-6个 | 2-3秒/段 |
| 30秒 | 6-8个 | 3-5秒/段 |

### 情绪节奏选择

| 类型 | 曲线 | 适用场景 |
|------|------|---------|
| 蓄力型 | 平稳→微变→加速→爆发 | 情绪积累 |
| 爆发型 | 沉默→暗涌→骤变→释放 | 冲击感 |
| 平推型 | 均匀递增 | 压迫感 |
| 消退型 | 高潮→缓降→余韵 | 诗意结尾 |

### 剧情密度原则

**每个分镜必须包含：**
```
[时间节点] [人物动作/位置变化] + [环境/道具变化] + [光线/色调变化]
```

## STEP 2 · MJ 生成参考图

### 比例选择（小红书）

| 内容类型 | --ar 值 |
|----------|---------|
| 竖版视频封面（首选） | `--ar 4:5` |
| 竖版视频素材 | `--ar 9:16` |
| 方形图文封面 | `--ar 1:1` |

### 白梦客美学配方

**通用线（故事/情绪短片）**
```
融合：胡金铨 + 王家卫 + 北野武 + 李安
色调：大地色50% + 苍青20% + 琥珀霓虹20% + 饱和红10%
光影：侧逆光+烟雾70% + 霓虹漫射30%
构图：前景遮挡+极简留白
节奏：长镜头留白70% + 抽帧慢动作20% + 硬切断刀10%
质感：Kodak Ektachrome 1970s color science
```

**高定线（高端品牌广告）**
```
融合：Apple + Chanel/Dior + 北野武 + 李安
色调：黑白40% + 大地灰20% + 冷白20% + 克制金/铜20%
光影：自然光50% + 单一致光源30% + 极轻霓虹20%
节奏：长镜头凝视60% + 静默留白30% + 偶尔快切10%
质感：16mm/35mm胶片，过曝1/3档，轻颗粒
```

### MJ 提示词结构

```
[美学参考], [画家参考]

[主体描述：人物/静物/风景]
[场景描述：环境、道具、氛围]
[光线描述：类型、方向、色温]
[色彩描述：主色调、撞色关系]
[构图描述：景别、留白、前景遮挡]
```

| 北野武蓝灰 | 400-500 |

## STEP 3 · Seedance 分镜提示词

### 核心原则

**视频模型是"视觉序列匹配器"。**
- 它理解画面 A→B→C 的变化
- 不理解因果逻辑和情绪标签
- 提示词要写成它能匹配的"视觉序列描述"

### 禁止项

```
❌ 任何文字：中文、英文、logo、水印、字幕
❌ 直接的情绪标签：紧张、愤怒、悲伤、快乐
❌ 抽象概念：自由、希望、梦想、时间
❌ 模糊的因果：因为他很紧张所以发抖
❌ 未量化的变化：灯光变亮、声音变大、速度变快
```

### 提示词书写规则

**写看到的，不写想到的：**
```
❌ 紧张（模型不懂这是什么画面）
✅ 手指在方向盘上收紧，关节处发白；额头有汗珠滑下

❌ 她感到绝望
✅ 她的眼睛慢慢从画面中心移开，落在某个空白处
```

**视觉变化 > 静态描述：**
```
❌ 灯光变冷
✅ 光从暖金变成冷蓝，被她的影子压过去，阴影边缘变得锐利

❌ 蓝焰喷射
✅ 幽蓝火焰从排气口喷涌而出，从细变粗，焰心比边缘亮3档
```

**环境变化是情绪的外化：**
```
人物离开走进维修区 → 光从暖金变成冷蓝
人物踩下油门 → 车身震动，背景快速后退
```

### 分镜提示词模板

```
【分镜N｜时长】
挂载：@角色图 / @场景图
Audio：[音效/BGM描述，含时间节点]
FORMAT：[时长] / [镜头数] / [对话] / [画幅]
STYLE：[影调] + [机型] + [质感代号]

【本分镜提示词】
[时间] [画面描述——具体的视觉元素和变化]
[时间] [画面描述——具体的视觉元素和变化]
[时间] [画面描述——具体的视觉元素和变化]

【镜头核心】[1句话：这场戏最值钱的是什么]
【目标物Must-Show】[必须出现的元素，逗号分隔]

禁止：任何文字、字幕、LOGO或水印
```

### 多参挂载原则

```
图片提供：人物外观、场景基调、静态构图、光线质感
文字必须补充：表情的微小变化、相机运动和视角变化、
              光影的动态变化（从哪到哪）、物体的运动轨迹
```

### 续镜衔接

每段结尾要留有承接点：
```
分镜01结尾：德比斯的手离开按钮
分镜02开头：按钮已经被按下（续接）

分镜01结尾：蓝焰吞没车身
分镜02开头：蓝光继续蔓延（叠化）
```

## STEP 4 · 视频生成（Seedance 2.0）

### 分段生成策略

超过 15 秒的镜头建议分段生成：
```
原始需求：15秒完整镜头
分段策略：
  分镜01：0-5s（建立场景+引入）
  分镜02：5-10s（主体动作）
  分镜03：10-15s（高潮/收尾）
```

**分段原则：**
- 每段 5-8 秒为宜
- 每段结尾要有承接点
- 使用相同 STYLE 保持一致性

### 生成后检查

```
□ 画面是否连续（没有跳帧）
□ 运动是否符合物理规律（没有违反重力）
□ 光影是否一致（光源位置、色温）
□ 角色一致性（人物外观是否保持）
□ 情绪外化是否自然
□ 是否有塑料拼贴感（边缘是否自然）
□ 信息密度是否合理（不是静态定格）
```

## STEP 5 · 剪辑合成

### 剪辑节奏参考

| 剪辑节奏 | 声音策略 |
|---------|---------|
| 快剪(1-3秒/镜) | 音乐节奏强、音效密集 |
| 正常剪(3-8秒/镜) | 声音正常、留白机会多 |
| 慢剪(8秒+/镜) | 声音简化、大量留白 |

### 声音回来时机

```
□ 情绪顶点后立即回来 = 延续紧张
□ 情绪顶点后留白2-3秒 = 让观众消化
□ 情绪顶点后留白5秒+ = 诗意/仪式感
□ 下一镜开始才回来 = 情绪独立，硬切感
```

## STEP 6 · 小红书发布

### 发布参数

| 参数 | 推荐值 |
|------|--------|
| 最佳时段（工作日） | 18:00-20:00 |
| 最佳时段（周末） | 10:00-11:30 / 15:00-17:00 |
| 发布频率（冷启动） | 每周3-4条 |
| 发布频率（成长期） | 每周5-7条 |

### 固定标签（每条必带）

```
#白梦客 #AI视频 #AI短片 #AI创作
#Midjourney #即梦 #Seedance
```

### 风格标签

```
#王家卫风格 #胡金铨美学 #北野武
#电影感 #氛围感 #东方美学 #新中式
#AI电影感 #AI短片 #情绪短片
```

### 蹭流标签

```
#AI #人工智能 #AI工具 #AI教程
#小红书运营 #内容创作 #涨粉
```

### 标题公式

```
公式1：场景+情绪 → 「xxx的xxx，xxx」
公式2：反差+悬念 → 「xxx但xxx」
公式3：数字+利益 → 「x招教你xxx」
公式4：身份+共鸣 → 「作为xxx，我xxx」
```

### 数据复盘（发布后48小时）

```
【内容诊断】
□ 标题吸睛（播放高但其他低 = 标题好但内容一般）
□ 封面吸引（播放低 = 封面/标题需优化）
□ 内容共鸣（收藏高 = 有价值内容）
□ 话题热度（蹭流类效果差 = 热点选错了）
□ 时段问题（播放低 = 换个发布时间试试）
```

```

- 主题：【xxx热点/情绪/品牌】
```
