# 广告提示词矩阵系统

## 功能

接收创意简报，生成针对不同平台优化的 AI 视频提示词。

## 输入参数矩阵

| 参数 | 选项 | 说明 |
|------|------|------|
| 平台 | Seedance 2.0 / Vidu Q2/Q3 / 海螺 | 决定语言和格式 |
| 时长 | 5s / 10s / 15s / 30s | 决定镜头数量 |
| 风格线 | 通用线 / 高定线 | 决定色调/节奏/光影 |
| 场景类型 | 开场/发展/高潮/结尾 | 决定叙事结构 |
| 运镜复杂度 | 简单/中等/复杂 | 决定camera指令密度 |
| 情绪 | 克制/平静/温暖/强烈 | 决定光影和节奏 |
| 行业 | 见ad-taxonomy | 决定具体技能调用 |

## 平台格式化规则

### Seedance 2.0 (即梦)

#### 基础参数
- **语言**: 中文优先（即梦是国产，中文理解更好）
- **时长**: 5-15s 单镜头，15-30s 多镜头
- **连接符**: `then cut to:` 或 `;`
- **时间戳**: 必须使用 `[00:00-00:05]` 格式
- **风格词**: 放在开头或用 `in the style of`
- **多模态引用**: 使用 `@` 语法，最多9张图+3视频+3音频

#### 核心结构（分镜段落式）

```
[电影风格]: [导演/风格参考]，[情感基调]，[质感描述]。

[核心对白/情感控制（可选）]: "xxx" — 用于AI理解情绪走向

[视频时长]: X秒
[脚本]:

[00:00-00:04] 镜头1: [景别/类型]。
场景: [具体场景描述]。
动作: [主体动作]。[细节补充]。
[灯光/氛围]: [光影描述]。
对白情感映射: [情绪外化方式]。

[00:04-00:08] 镜头2: ...
```

#### 导演风格速查

| 导演 | 风格关键词 | 适用场景 |
|------|-----------|---------|
| 王家卫 | 抽帧，慢动作，霓虹，黄绿偏色，颗粒感，潮湿 | 情感/孤独/都市 |
| 北野武 | 静默，长镜头凝视，蓝灰冷调，硬切断 | 暴力/极简/萧索 |
| 胡金铨 | 大地色，烟雾散射，前景遮挡，留白 | 武侠/古典/禅意 |
| 丹尼斯·维伦纽瓦 | IMAX 70mm，颗粒写实，史诗规模，低饱和 | 科幻/沙漠/恐惧 |
| 李安 | 隐忍，东方式意境，暖黄，自然光 | 家庭/情感/东方 |

#### 人脸限制与绕过
1. 使用 `@` 引用真人图片作为参考（图生视频）
2. 负向提示词: `negative: realistic human face, photorealistic portrait`
3. 风格化描述: `in the style of 1970s film still, cinematic portrait`
4. 局部特写: 眼部、手部、侧脸等不完整人脸
5. 艺术化处理: 剪影、侧影、光影遮脸

### Vidu Q2/Q3

- **语言**: 中英混合均可
- **时长**: Q2: 2-8s, Q3: 1-16s
- **风格词**: 必须放在提示词最前面
- **连接符**: `|` 或 `→`
- **运动幅度**: `--movement-amplitude` 参数（1-10）
- **角色一致性**: 1-7张参考图维持角色
- **模式**: `pro`(质量优先) 或 `speed`(速度优先)
- **镜头格式**:
  ```
  [Style word] [场景描述],
  [主体] [动作],
  [运镜] [光影],
  [氛围]
  ```

### 海螺（图像转视频）

- **语言**: 中文
- **时长**: 5s 限制
- **核心**: 基于输入图像，描述动态预期
- **优势**: 相机控制强(推/拉/摇)，擅长情感微表情
- **降失败率**: 图生视频比文生视频失败率低很多
- **负向提示**: 用自然语言描述不要的元素
- **镜头格式**:
  ```
  [图像内容],
  [期望的动态]: [动作描述],
  [光影变化],
  [氛围延续]
  ```

## 提示词长度与结构策略

### 核心发现

**结构化分段提示词 >> 长散文 >> 简短提示词**

| 时长 | 推荐方式 | 长度 | 示例 |
|------|----------|------|------|
| 5s | 简洁描述 | 50-150字 | 单镜头核心动作 |
| 10s | 适度详细 | 150-300字 | 1-2个镜头分段 |
| 15s | 完整结构 | 400-800字 | 3-4镜头时间戳分段 |

### 提示词质量层次（5层结构）

| 层次 | 内容 | 示例 |
|------|------|------|
| 1. 主体 | 人/物/场景 | 穿红裙的女子 |
| 2. 风格 | 视觉风格 | 王家卫电影风格，霓虹色调 |
| 3. 动作 | 动态描述 | 在雨夜街头低头快步走 |
| 4. 光影 | 灯光氛围 | 侧逆光，湿润感 |
| 5. 情绪 | 氛围基调 | 孤独疏离感 |

### 专业术语效果

**有效术语 vs 无效描述：**

| 无效（空洞） | 有效（专业） |
|--------------|--------------|
| 电影感 | 2.35:1宽银幕，24fps，胶片颗粒 |
| 好看 | 侧逆光打出发丝光，浅景深虚化 |
| 漂亮的光 | 丁达尔效应，体积光，Lens flare |
| 动态 | 环绕摇镜，缓推镜头，跟拍 |

### 时间戳分段原则

15秒视频分段示例：
```
[00:00-00:03] 远景建立：俯拍雨夜城市全景
[00:03-00:06] 中景引入：女子背影在霓虹灯下
[00:06-00:10] 近景情绪：面部特写，低头快步
[00:10-00:13] 动作高潮：转身回眸，慢动作
[00:13-00:15] 落版：渐隐fade to black
```

### 导演风格引用（有效捷径）

用导演名代替大量描述：

| 导演 | 代表风格 |
|------|----------|
| 王家卫 | 霓虹色调，慢动作，孤独情绪 |
| 胡金铨 | 70年代胶片，留白，武侠美学 |
| 北野武 | 静默长镜头，冷峻色调，暴力美学 |
| 李安 | 东方意境，饮食男女，温情叙事 |

**用法**: `王家卫风格，霓虹色调，手持跟拍`

## 动作分解原则

复杂动作需拆分为子动作（每个5s内）：

| 复杂动作 | 分解示例 |
|----------|----------|
| 起床 | 1.手伸向闹钟 2.人坐起 3.脚踩地板 |
| 行走 | 1.抬脚 2.迈步 3.落地 4.重心转移 |
| 喝水 | 1.手握杯 2.举杯 3.倾斜 4.放下 |

## 负向提示词模板

```
Negative prompt:
- 无水印/字幕/Logo
- 无模糊/抖动
- 无过度饱和
- 禁止XXX元素
```

## 常见错误

1. **模糊描述**: "一个人在走" → "穿红风衣的女人在雨夜霓虹街头低头快步走"
2. **内容过多**: 15s内不要尝试完整故事弧，分镜
3. **手部特写**: AI对人手处理差，避免手部近景，如需手部用参考图
4. **风格冲突**: 不要混用矛盾美学（如赛博朋克+田园）
5. **忽略负向提示**: 总是指定要避免的元素
6. **真人人脸**: Seedance默认生成会变形，用 `@` 引用真人图 + 负向提示词绕过

## 输出格式

screenwriter/director 使用此技能时：

1. **解析输入矩阵**: 平台 / 时长 / 风格线 / 场景类型 / 运镜复杂度 / 情绪 / 行业
2. **选择美学配方**: 根据行业和风格线确定色调/光影/节奏/质感
3. **生成平台化提示词**: 按对应平台的格式化规则输出
4. **引用参考案例**: 在提示词末尾标注参考案例

## 使用流程

```
输入 → 解析参数矩阵 → 选择技能(如perfume-luxury) → 应用美学配方 → 生成平台化提示词 → 输出
```

## 提示词质量检查清单

- [ ] 平台语言正确（即梦中文，Vidu风格前置，海螺中文）
- [ ] 时长与镜头数量匹配（5s=1镜头，10s=1-2镜头，15s=2-3镜头，30s=3-4镜头）
- [ ] 色调参数完整（主色+辅助色+点缀色）
- [ ] 光影描述清晰（光源类型+方向+氛围）
- [ ] 运镜指令具体（movement + angle + subject）
- [ ] 参考案例标注
- [ ] 无产品直接展示（高定线要求）
- [ ] 节奏分配合理（长镜头/快切比例）
