# 广告分类 & 技能路由

## 功能

收到 Brief 后，根据以下规则识别广告类型并路由到对应技能。

## 识别流程

1. **解析 Brief** — 提取行业关键词、产品类型、风格暗示
2. **匹配行业** — 根据产品判断所属行业
3. **匹配风格** — 根据客户/品牌/Brief 描述判断风格
4. **路由技能** — 输出对应技能名称

## 输出格式

```
行业：[识别结果]
风格：[识别结果]
推荐技能：[技能名称]
理由：[简要说明为什么选择这个技能]
```

## 未知类型处理

如果 Brief 无法明确分类：
- 输出"需要更多creative-director人工判断"
- 列出最可能的 2-3 个选项供用户选择

## 提示词生成流程

路由到对应技能后，使用 `prompt-matrix` 技能生成最终提示词：

1. **输入**: 平台 + 时长 + 风格线 + 场景类型 + 运镜复杂度 + 情绪 + 行业
2. **解析**: 从对应技能读取美学配方（色调/光影/节奏/质感）
3. **生成**: 按 prompt-matrix 模板生成平台化提示词
4. **输出**: 完整 AI 视频提示词
