### 1. 成功模式捕获

**格式**：
```

适用场景: <何时使用>
有效性: <高/中/低>
```

### 2. 失败/问题模式

本次项目中遇到了哪些问题？问题的根因是什么？

**格式**：
```
问题描述: <具体描述>
涉及技能: <技能名>
失败原因: <根因分析>
建议修复方向: <如何改进>
```
