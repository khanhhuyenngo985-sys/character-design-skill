# 小红书内容运营技能 · 白梦客版

> 版本：v1.2 | 更新：2026-04-07 | 定位：热点抓取 + 内容创意生成

---

## 核心功能

1. **热点抓取** — 多平台数据汇总，小红书热门话题、标签、趋势
2. **内容创意生成** — 基于热点输出可执行的内容方向
3. **标题文案建议** — 爆款标题模板 + 标签推荐
4. **MJ提示词初稿** — 配合 mj-prompt 技能生成封面图
5. **竞品分析** — 同类型账号爆款内容追踪
6. **数据复盘** — 内容表现追踪与优化建议

### last30days 研究引擎（核心工具）

**`/last30days`** 是跨平台热点研究的核心引擎，可搜索：
- Reddit、X/Twitter、YouTube、TikTok、Instagram
- Hacker News、Polymarket 预测市场
- Web 网页搜索

**使用方式：**
```
/last30days AI视频创作趋势
/last30days 小红书 AI短片 热门话题
/last30days AI广告 复刻 创意
/last30days 情绪短片 电影感 热门
```

**研究完成后AI自动输出：**
- 带引用的热点分析报告
- 各平台热度数据（ upvotes、likes、views）
- 社区讨论的核心趋势
- 可直接用于内容方向的建议

### 热点类型优先级

| 优先级 | 类型 | 示例 |
|--------|------|------|
| P0 爆发期 | 正在热炒的话题 | 热搜前10 |
| P1 预热期 | 刚兴起的话题 | 热搜11-50 |
| P2 稳定期 | 持续发酵的长尾 | 话题标签 |

### 热点筛选标准

- 与**影像创作**相关度
- 有**情绪共鸣点**
- 可以用**AI短片**呈现
- 避开敏感词（政治、擦边、虚假医疗等）

### AI辅助热点分析

当没有明确热点时，用 last30days 分析：

```
/last30days AI视频创作 热门趋势 2026
```

或直接告诉AI：

```
帮我分析当前值得蹭的热点，要求：
1. 列出3个正在热炒的话题（带热度数据）
2. 每个话题的受众画像
3. 用AI短片呈现的可行性评估
4. 给出3个内容方向建议（带标题初稿）
```

## 七、执行工作流

### 每日/每周热点抓取流程

```
1. last30days热点研究（5分钟）
   → /last30days AI视频创作 热门趋势 2026
   → /last30days 小红书 AI短片 热门话题
   → 产出：跨平台热点分析报告

2. 热点筛选匹配（3分钟）
   → 根据报告选出2-3个适合白梦客的热点
   → 评估：情绪共鸣度 + AI短片可呈现性

3. 生成内容方案（5分钟）
   → AI输出：标题+文案+MJ提示词+标签
   → 配合 mj-prompt 技能生成封面图

4. 制作内容
   → MJ出图（4张精选1张）→ Seedance动画 → 剪辑

5. 发布+记录
   → 按模板记录，48小时后复盘数据
```

### 快速执行模板

当你想做一条内容时，告诉AI：

```
我要做一条：
- 主题：【xxx热点/情绪/品牌】
- 方向：【情绪短片/热点蹭流/品牌复刻】
- 时长：【10秒/30秒/1分钟】
- 比例：【4:5封面 / 9:16视频】
```

AI自动输出：
- 标题选项（3个）
- 文案初稿
- MJ提示词（用白梦客美学配方）
- 标签建议
- 发布时间建议
